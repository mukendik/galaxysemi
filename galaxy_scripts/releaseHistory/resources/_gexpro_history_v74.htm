V7.4.0 (1 July 2015) (Top)
Enhancements
Case Number Subject Description
GCORE-3694 Add a search option to Interactive Tables Examinator already includes a filter option for Interactive Tables, which shows the tests that match the search criteria and hides all the other tests. The search option will find the tests that match the search criteria with all other tests visible.
GCORE-3079 Allow the use of channel name when using the js option to set the name for MPR tests When analyzing MPR tests, it is useful to include the channel name in test names in addition to physical name.
Bug fixes
Case Number Subject Description
GCORE-4251 IFF data parser produces a yield/fail count discrepancy IFF data parser produces a yield/fail count discrepancy when user sets the Parameter Pass/Fail Rule to Use Pass/Fail Flag if Valid. Some Examinator reports (e.g. Test Failure Pareto, Interactive Table, and Interactive Histograms) show failures but distribution statistics show zero failures.
GCORE-4184 The deviceid and part_id from the IFF data parser do not match From run 1, part_id = 0 and devid = 1. Expected result is part_id = devid.
GCORE-4010 When a user deletes an option from the admin options window, it is no longer visible in options list When a user deletes an option, the option is no longer visible in the list of options. To make the deleted optoion re-appear in the list of options, the user must close the window and re-open it.
GCORE-3857 When performing Gage Analysis using Report options->Significant digits, the results are truncated User is having problems performing Gage R&R analysis. When user selects "Significant digits" from the Report Options, the results are truncated.
GCORE-2173 When comparing two files with MPRs, the MPR tests are duplicated in test pick list in Interactive Charts The bug occurs when comparing two files with MPRs with the option selected to 'not merge' the MPRs. When the user then goes to the picklist of tests in the Interactive Charts, each of the MPR tests appear twice on the list.
GCORE-1596 Examinator status message says "Building Advanced Reports..." when the report is done After changing the bin color of the wafer map, Examinator prompted the user to "rebuild the report." User clicked "ok" and successfully built the report, but there is a status message at the bottom of the Examinator GUI saying "Building Advanced Reports..." when building is done and the report is ready for use.
GCORE-910 It takes too long to open the filter pick list from the interactive tables In the interactive tables, there is a funnel icon that is used to filter tests which are displayed. When a user clicks on the funnel icon there is an unacceptable delay before the test list window appears. Users expect to immediately see the test list window after clicking on the funnel icon.

V7.4.RC1 (2 June 2015) (Top)
Bug fixes
Case Number Subject Description
GCORE-3960 V7.3 Characterization Report is different from V7.2 Characterization Report V7.3 Characterization Report is different from V7.2 Characterization Report when inserting the same set of data files into a V7.3 TDR (Test Data Repository) and into a V7.2 TDR.
GCORE-3892 Error executing SQL query of E-test (PCM) test data Examinator-Pro generates an "Error executing SQL query" error message when the user queries the database for E-test (PCM) data and attempts to build a report.
GCORE-3891 User cannot see any data when analyzing IFF final test data files If the user converts the IFF file to STDF in the Examinator toolbox, the Examinator report shows a test list with zero executions. Further examination of the STDF by dumping it to ascii shows that there are PIR/PRR record pairs and PTR records between the PIR/PRR record pairs with valid results.
GCORE-3890 Examinator crashes after building a report The crash is triggered when the user performs the following actions:
1. Query the database
2. Build a report
3. Go to interactive tables
GCORE-3887 Limits scaling is not correct for IFF data files For data parsed from IFF data files, the units and scaling factors do not follow the STDF standard. As a consequence, Examinator-Pro produces bad results when results are scaled.
GCORE-3861 Examinator does not generate a wafer map from an "inline" e142 XML data file When loading an e142 XML data file that is in one line, no wafer map is generated.
GCORE-3845 IFF data parser is extracting the wrong wafer ID The IFF parser assigns WAFER_NUMBER to the wafer_id field in the Test Data Repository (TDR), which is wrong. The IFF parser should assign WAFER_ID to the wafer_id field in the TDR, and WAFER_NUMBER can be ignored by the IFF parser.
GCORE-3721 Insertion of EDF file causes Examinator and Yield-Man to crash This EDF file causes many crashes. For instance, making a default Examinator report causes Examinator to crash.
GCORE-3697 Merging test and retest does not create a PCR record When merging test and retest data, there is no PCR record created from the original data files.
GCORE-3687 Final Test LVM FETTEST data is incorrectly processed as wafer sort data In Examinator, Final Test (FT) LVM FETTEST data is processed as final test data, as expected. But when Examinator parses FT LVM FETTEST data in a PROMIS data file, it rejects this data with the error message "wafer not found in PROMIS info file."
GCORE-3660 Shifted/missing data after creating a CSV report within the Characterization Wizard User queried the Characterization database. When the user created the CSV report, the test condition's data was missing; it was shifted. The end result is wrong data in the CSV report.
GCORE-3252 Invalid start time causes a parser error When reading a data file that has an invalid start time, the parser does not recognize the file type even though the overall format is good.
GCORE-1695 Test Name appears as "-?-" with no data User analyzed a CSV file in Examinator and noticed that a test with a valid test number has a question mark "-?-" for test name, and there is no data for the test. User confirmed that data exists for this test.
GCORE-1410 The markers are displayed incorrectly in a probability plot Displayed results in interactive Probability Plot do not match results in interactive Histogram Chart. Values which are ~6 sigma in the Histogram Chart are ~3 sigma in the Probability Plot.
GCORE-1367 Report Center reports incorrect bin names if bin names change over the life of the product If bin names for a particular product change over that product's life span, Report Center will report the first bin name instead of the latest bin name.
GCORE-1315 Examinator does not generate the wafer map on an "inline" G85 XML map The user could not generate a wafer map for a particular.XML file. This is likely due to the file containing inline data, which prevents Examinator from parsing it correctly. Inserting line breaks to separate the lines in the file allows the wafer map file to be generated.
GCORE-1239 Incorrect bin names appear in Report Center bin reports when splitting products with same bin number but different bin names When doing a Stacked Bin report at final test and splitting by Product ID, the soft bin names are all the same while they differ for each product. The report is expected to show the appropriate bin names.

Enhancements
Case Number Subject Description

In previous versions when datalogs were merged, the interactive table showed different entries for the same die XY making it harder to interpret the results.

Bug fixes
Case Number Subject Description
GCORE-3582 No auto-reconnect to MySQL server when connection is broken during a query When the connection between the Test Data Repository and MySQL server is broken during a query, automatically attempt reconnect and trace.
GCORE-3430 Histogram charts are not displayed when using the Histogram option within Interactive Tables The user was working with Interactive Tables and selected multiple pins from the data file. When the user then right-clicked and selected Histograms from the menu, Examinator brought up Interactive Charts with no graphic.
GCORE-3213 Wrong kurtosis and skew values reported on interactive table when splitting data by test site When analyzing multi-site data and splitting the data set by test site, the kurtosis and skew values reported on the interactive table (for each site) are from the first test site. On the static table, kurtosis and skew values are correct for each test site.
GCORE-3197 Display multiple test limits as markers in interactive charts Display multiple test limits as markers in interactive charts with proper scaling for all the test limits.
GCORE-3191 If ATDF file is missing an MRR, the toolbox should not try to convert the last record The toolbox expects the last record in an ATDF file to be an MRR record. When an ATDF file is missing an MRR, the toolbox should ignore the last record in the file instead of converting it, which will cause problems.
