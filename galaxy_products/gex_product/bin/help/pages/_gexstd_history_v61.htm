V6.1 B155 (22 December 2009) (Top)
Examinator:
  • Case 2456 - Enhanced Histograms and Box Plots: Added new functionality to enhance visual appeal and functionality. Limit colors match group colors on histograms and box plots.
  • Case 2441 - Virtual Test Creator: Quantix now offers the ability to create 'virtual' tests on the fly using data from existing tests. For example, it is now possible to perform a mathmatical operation on two or more existing measurement results, and report those results in Quantix using all of the available analysis tools such as statistical summaries, charts, and wafermaps.
  • Case 2362 - TEL Prober Files: Added the capability parse and analyze TEL wafer probe files.
  • Case 2037 - Multiple partial data files: Added a capability to accept multiple files from the same wafer, and stich them together into a coherent gouping for logical analysis. This capability is useful when partial wafer files are typical, as might be the case for test programs with very long test times.
  • Case 2395 - Revised Database Administration Page: The Database Administration page has been updated to include additional valuable information such as 'Connected', 'Db version', and 'Update Status.'
  • Case 2306 - Grace Semiconductor Parser: New capabilities for parsing and mangement of files from Grace Semiconductor
  • Case 2295 - Silan Fab Parser: New capabilities for parsing and mangement of files from Hangzhou Silan Microelectronics.
  • Case 2260 - Enhanced capabilities for dealing with consolidated or re-tested data: Added a "Consolidated" drop down box to the Data tab. When set to Consolidated data, Quantix will automatically filter die that are retested from first pass results. Quantix will also filter data from subsequently re-tested die. The new capability allows for analysis of yield exclusive of re-tested die.
  • Case 2255 - New Command Line Capability: Added the capability to launch Quantix from the command line while calling multiple files. This new capability expands the flexibility of Quantix to work seamlessley with other programs & scripts that may call on Quantix for analysis capabilities.
  • Case 2236 - Modified Default Configuration: Modified and enhanced the default user configuration to more closely align with user expectations.
  • Case 2233 - Sipex Parser: Added a parser for Sipex test data.
  • Case 2196 - Enhanced Global Info Report Tab: Added an option to the �Options� tab called �Global Info�, which allows user to choose a summary or a detailed Global Info report when merging or comparing groups of files (the default value is �Summary�).
  • Case 2179 - New What-If Analysis: Added a column to the What-if table to display Failing Bin number.
  • Case 2178 - Improved Options Menu Navigation: Added context menu to the �Options� tab to expand/collapse all submenus or just a selected submenu.
  • Case 2177 - Enhanced 3D Interactive Wafermap 'Find' Utility: Added a feature to the 3D Interactive Wafermap that maintains a Find criteria ("Find part(PART_ID)", "Find Die: X,Y") so that once called, the criteria are persistent until changed for an analysis session.
  • Case 2163 - Improved Multi-Wafer Wafermap Analysis: Added a feature to the Bin Wafermap analysis tool that will maintin contstant color codes across wafermaps during an alalysis session. This allows for quick comparison of bin wafermaps.
  • Case 2162 - Improved wafer flat detection and alarms: Quantix will now give an alarm if the software detects that the flat represented by the data is different from what is specified in the data file. When this happens, Quantix will now highlight the line in RED if either the detected flat location is different from the one specified in the STDF file or there is no flat location specified in the STDF file.
  • Case 2158, 1778, & 1625 - Automatic STDF repair utility: Quantix will now, at the users option, offer to attempt a repair of a corrupt STDF fiile. If successful, the file can be analyzed as if the file were not corrupted.
  • Case 2146 - Interactive Window Statistics Tab: Added additional functionality to the Interactive Window Statistics Tab. Min/Max and Specification limits.
  • Case 2117 - Enhanced FTR pass fail reporting: Added the option to specify how Quantix will handle multiple FTR results. The new option will specify if Quantix should fail if any runs fail, or fail the die only if all runs fail.
  • Case 1991 - Wavemaster Parser: Added the capability to parse and manage Wavemaster test data.
  • Case 1985 - Optical parsers: added the capability to parse and manage a suite of optical test data formats, including ImageMaster, Diffractive Test, Micrulupi, & QV302 testers.
  • Case 1570 - Die to Die Mismatch Wafermap: Added a new wafermap comparison tool. This new tool displays color codes correspoding to software or hardware bin values for two wafermaps. Identical bins are not colored. Changed bins are identified with two colors for 'before' and 'after.'
  • Case 2498 - Optical file format: Fixed a bug in reading Row and Column for optical devices. In some cases, X and Y coordinates would be reversed. Added logic to prevent this from occurring.
  • Case 2493 - Data not showing up for tests not included with reference dataset: Added the capability to display test data from tests that are not part of the reference data set. For some types of analysis, particularly comparing files are datasets, Examinator would not display data from subordinate datasets if the test numbers or test names were not part of the reference dataset.
  • Case 2475 - What if not calculating delta yields correctly with trial and error approach: Fixed a problem that would cause the Delta Yield column on the What-If yield analysis to recalculate improperly under certain conditions.
  • Case 2440 - Test limits do not display when comparing datasets: Fixed a problem with interactive charting that caused test limits to display improperly, or to not display at all, under certain conditions.
  • Case 2414 - Program Crashing: Fixed a problem that would cause Quantix Examinator to crash under very specific circumstances with a particular data set.
  • Case 2430 - First page of a Quantix Characterization report: Fixed the report generation tool so that it once again accepts html tags and text for the report first page. This capability applies to static reports, such as PDF, PPT, and Word Docs.
  • Case 2222 - Custom home page content not correctly displaying: Enhanced the software's capability to detect and properly display html pages.
  • Case 2442 - Save button for Enterprise report script does not actually save files: Fixed a bug that prevented the 'Save Script' button from properly saving a script.
