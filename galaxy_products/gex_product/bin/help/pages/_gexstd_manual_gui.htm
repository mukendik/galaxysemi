User guide: The Examinator Interface

The Quantix Examinator interface has been designed to work like a web site: it relies on the logical approach of actions triggered by clicking on links. Nowadays, web browsing has become quite natural to all of us--technical as well as non-technical people!

So it made great deal of sense to have the Quantix Examinator use this approach since our mission is to provide users with a very easy, yet powerful solution.

As with any web interface, the home page is displayed at the application launch time and includes links to the most important resources. As with any web browser, you have the navigation buttons and the list of visited pages that can be selected at any time. Use the navigation bar on top of the Examinator interface if you want to go back to a previous page.

From this navigation bar you can initiate the following actions:

  • Back arrow: reload a page previously visited

  • Next arrow: return to pages visited after using the back arrow

  • Stop button: abort any data analysis and script execution

  • Home button: reload the home page (same as clicking the Home tab)

  • Script button: go to the Scripting center page

  • Printer button: print the HTML page currently visited (can be the home page, any report page, or any help page). If you have Adobe Acrobat installed, you can redirect the printer output to generate .PDF files from your HTML pages.

  • Snapshot / Copy the content of the Examinator window's content into the clipboard, an excellent way to cut/paste information between applications such as Microsoft PowerPoint or Word.

  • Tutorial videos: launch the Multimedia player with Examinator tutorials and presentations (Windows version only)

  • Context help: Launches the Netpus character to provide you with context help.
