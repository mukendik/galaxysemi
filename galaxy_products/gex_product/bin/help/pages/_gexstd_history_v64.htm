V6.4.16 (4 May 2012) release history (Top)
Examinator:
  • Case 6132 - G85 ASCII Export contains redundant symbol: When exporting a map to a G85 ASCII format, we sometimes get the same ASCII symbol for different bins. This prevents the user from identifying the bins on the exported file.
  • Case 6127 - GexPAT: No Data Samples for results at first access: When accessing PAT Report, get No Data Samples for results.
  • Case 6094 - Crash in Scatter chart when multi dataset: Crash in scatter charts when multiple datasets (compre files, compare datasets,...).
  • Case 6060 - can't see interactive wafer map / strip map: user cannot see interactive wafer map / strip map. static map works fine.
  • Case 6005 - Crash in interactive chart while picking test list: after a Pat processing, in interactive chart, crash when trying to pick a test if rebuilt test list.
  • Case 5946 - GexPAT: T value in recipe causing different distribution shape results: GexPAT gives different results - different distribution shapes - for tests when Smart & Adaptive is active and there is a number in the T field.
  • Case 5887 - Can't get static Advanced chart in more reports: When splitting on one parameter, if for the first splitted parameter there are no valid execs, we can get interactive chart, but can't get the same in static reports (More Reports).
  • Case 5834 - converting csv file to stdf with custom parser gives run_id =0 when inserted the stdf into db: take a csv and convert to stdf using a custom parser created with the gex parsing wizard. then insert the resulting stdf into the database and see that xx_run.run_id = 0 for all parts.
  • Case 5680 - Can't get the interactive softbin map. Regression over 6.3: The interactive SoftBin Wafer map of the attached CSV file shows up well in V6.3, while nothing shows up in v6.4. It's worthy to note that no DIE_X/DIE_Y are filled in the CSV file.
  • Case 5289 - interactive wafer map / strip map won't display if there are no x/y coords in the csv: if the Quantix csv file has no values in the x and y coordinate columns, the strip map will not appear in the interactive section although it will appear in the static section.
  • Case 5014 - miscellaneous stats in pat report are reporting non-sense values: after running a pat report, the misc stats column shows invalid values.
