Data Analysis Settings

From the settings page, you can decide which report sections to create or how to drill into the data. Depending which GEX module you are allowed to run (depending on your license contract), you may have access to some or all of the following selections:

  • Instant Report
  • Data Mining - Yield Analysis

 

GEX-Instant Report selection offers: 

  • Global information: tells you the files processed, date of creation, lot ID, etc..

  • Test statistics: a table with statistical information (execution count, Mean, Range, Cp, Cpk,...)

  • Pareto: Pareto tables for test results (Cpk), test failures, binning results

  • Binning summary: list of software and hardware binning

From the More reports selection, you can have more advanced reports such as sharp histograms, trend charts, scatter XY plots, etc..
.

GEX-Data Mining selection offers: 

  •   Guard banding drill to play on your test limits or Cp/Cpk and see the impact on your yield. Once you have tried different combinations and reached the one you like, simply take a snapshot (using the camera icon ) so information are saved to the HTML report! It's that easy. 

For more information, see the Data Mining - Yield Analysis page.
