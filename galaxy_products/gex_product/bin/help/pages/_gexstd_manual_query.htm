'Data' Tab: Datasets selection

Once you have selected a specific assistant from the home page, the Data page is displayed. Its layout will vary slightly depending on the assistant selected. If you select the assistant to analyze a single Dataset, then the page will only let you fill the filter for a single query. If you select compare Datasets, you will be able to enter as many query definitions as you want, and all will then be compared

On the Data page, you'll find some of the following icons/fields:

Icon Description
Dataset Name Label used in all the reports to refer to your query. Enter a string that is meaningful (e.g: Summer Production)
From Database Lets you select the database on which the query has to be executed: data resulting from the query are read from the database (data collection) you select here.
Time Period Lets you define the time period filter of your query. E.g: 'Today' will only retrieve data of the day.

(New query)		 When comparing queries: click this icon to create a query.


(Query properties)

When comparing queries: click this icon to edit/change the query currently selected in the Query list.

(Remove query)		 When comparing queries: click this icon to remove a query from the list

Filters

A query is nothing more than a request over a data collection with a filter criteria. Examinator offers easy filtering on many fields such as:

  • Lot ID

  • Process ID

  • Operator name

  • Facility ID

  • Floor ID

  • etc.
