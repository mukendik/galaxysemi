License Activation

Examinator ToolBox supports the following running modes:

  • Standalone: uses local CPU license

  • Client/Server: uses floating license

Laptop Users...

Evaluation mode
Select the 'Evaluation' running mode and get 10 days of full access to the Examinator ToolBox application. Then click the 'Next' button to start. After 10 days, you'll need to have placed an order to quantix_sales@mentor.com and have received your license file to continue using the software.


Standalone mode

Select the 'Standalone' running mode to start the process of activating the release for this specific computer. You'll then have to:

  1. Enter your full name

  2. Enter the Product Key (given to you when you placed your P.O.)

  3. Click the 'Next' button

  4. Email the file gex_request.txt created to gex.license@galaxysemi.com

  5. Within few minutes, you will receive your license file and activation instructions (email auto-reply)


Client/Server mode

Select the 'Connect to server' running mode to start the software and have it request a floating license from the Examinator ToolBox license server. You'll have to:

  1. Enter the license server name or IP address

  2. Enter the communication port # (Socket port#)

  3. Click the 'Next' button

Arguments at launch time

The Examinator ToolBox application supports launch time arguments that can be very convenient:
Line Description
gex -WTB Displays the Examinator ToolBox Welcome page. Lets you reset the running mode and/or server name+port number.

eg: gex -WTB will let you display the Examinator ToolBox welcome page and allows to select any running mode available.


GEX-LM License manager

The gex-lm application handles all the floating license management and authorization mechanisms. It must be running on the server prior to running any client application (Examinator ToolBox or GEX-LS). This application communicates with the other software using the TCP/IP protocol. The only setting you may want/need to change is the socket port number used for the communication channel. If GEX-LM is launched without any argument, the default socket port 4242 is used. To force the application to use a different port, simply launch the application as follows:

gex-lm -port <PortNumber>     where <PortNumber> is the decimal Socket port number to use.

eg: gex-lm -port 4243

GEX-LS License status

You can see the current license usage at any time launching the client application gex-ls. When executed, simply specify the server name (or IP address) that is running the gex-lm application, and the socket port number used to establish the communication (default is 4242 unless you have specified a custom number when launching the gex-lm application on the server).

 

Laptop users: installation prerequisites

When running Examinator ToolBox software for the first time, a license request file with the computer hardware identification profile is created (except if running in Server mode). If your laptop sometimes connects to a docking station, or if you have set different hardware profiles on your computer, it may happen that Examinator ToolBox refuses to run if you do not follow the correct license activation process. Here is the process you MUST follow to ensure a smooth Examinator ToolBox Standalone or Evaluation license activation on your laptop:

  1. Have the Examinator ToolBox software installed on your notebook (you can be using a docking station at this stage, it's not a problem)

  2. Disconnect your laptop from its docking station

  3. Log to your laptop and activate your off-line profile if you have one (so as to have your computer in the exact same situation as when working disconnected from your network)

  4. Run Examinator ToolBox , select the Standalone (or Evaluation) running mode and enter your Product Key ID then press the  Next button.

  5. Examinator ToolBox will create the gex_request.txt license request file with the correct hardware profile information

Note: If you didn't follow the above steps, simply restart from step one, so your previous gex_request.txt gets updated.
