Report tab

This page will display the last report generated, if any, or links to the assistant pages if no report has been generated yet.

You can build a new report at any time; all you need to do is click the Settings tab to edit the report sections you want to create/change, then click the Build report button again!

 

Use the toolbar to navigate into the report pages visited...and click the Snapshot / Copy icon at any time to copy the window content into the clipboard.
