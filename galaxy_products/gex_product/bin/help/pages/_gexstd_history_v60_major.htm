Examinator family : V6.0 Major Enhancements
(Examinator-Characterization, Examinator-Production, Examinator-PAT)



IMPORTANT NOTE:

Customers upgrading to V6.Y Quantix applications (Examinator-Characterization, Examinator-Production, Examinator-PAT, Yield-Man, PAT-Server), and connecting to a Gex-Lm license manager to get a license token, must make sure that the Gex-Lm license manager is also upgraded to V6.Y.


Examinator Version 6.0 has a new look with all buttons and screens improved for a clean and intuitive user interface. Interactive tables, charts, and wafermaps have been synchronized for seamless navigation between components.
Most of these improvements are a direct result of feedback and requests from customers.

TOP 10 enhancements (Top)
Interactive Charts
  • New - One click report generation from interactive charts.
    Make modifications to the dataset graphically, then re-create reports in real time with updated statistics, charts, paretos, etc.
  • New - "Multi Chart" tab displays 4 graphs on one page for a single test (histogram, trend graph, box plot, and new probability plot). A great way to quickly provide multiple perspectives of a specific test.
  • New - Global graphical outlier removal in the interactive histogram allows the option to remove all test data for the selected device/die. Results are reflected in the charts of all tests.
Interactive Wafermaps
  • Improved - Interactive drill down from histogram or trend chart to see corresponding die position on wafermap including data table for all tests. This allows you to graphically locate outliers or die with parametric issues.
  • Improved - Batch 3D parametric wafermaps for multiple tests. Includes sliders for manipulating high and low limits to your liking.
  • Improved - One click drill down in wafermap displays all tests results executed in the flow for the selected die.
What If / Virtual Retest tool
  • Improved - What If / Virtual retest tool has been re-written to improve speed and accuracy. You can also now save your limits file and import it back later. All changes retain the correct colors (yellow for "If" and green for "Then").
Interactive 2D/3D Tables section
  • New - Part Results table within the Tables section. This table will also allow you to select any die and it will show the results for all tests for that specific die.
Compare files with multi-site data
  • New - Multi-site compare displays per site results for both data sets.
What If / Virtual Retest tool
  • Improved - Toolbox - dump, repair, edit, and merge datafiles.
