7276 Display the name of the Options profile you are using and maybe the path Currently there is no way to really tell which Options profile you are currently using. You can see what the default profile is, but is that the one you are currently using? It would be helpful to show this. 7427 Options file selection - When you check the default checkbox, also select that row in the GUI When starting Examinator if there isn't a default Options file, a window will pop up to select the Options file you wish to use. If the default checkbox is selected next to the file, that row doesn't get selected and the file from the row previously selected gets loaded. From a user perspective, they think the one they selected with the check box will get loaded. Please also select the row of the file when the checkbox is selected.
