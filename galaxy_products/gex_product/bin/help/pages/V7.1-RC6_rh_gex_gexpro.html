7256 Request to export additional STDF field with Yield-Man CSV converter It is requested to be able to export additional STDF fields with the Yield-Man CSV conversion task. This includes standard STDF fields but also custom DTR fields.
7276 Display the name of the Options profile you are using and maybe the path Currently there is no way to really tell which Options profile you are currently using. You can see what the default profile is, but is that the one you are currently using? It would be helpful to show this. It may also be helpful to show the path of the options file, especially for Apps when asking for the user to send the options file, though the path might look ugly.
7335 Allow Global Options changes to the Manual Production TDR Customer can change the global option to allow duplicate test numbers in Yield Man, but cannot do the same in the Manual Production TDR, these Global Options are 'greyed out' and unavailable from Examinator Pro.
7427 Options file selection - When you check the default checkbox, also select that row in the GUI When starting Examinator if there isn't a default Options file, a window will pop up to select the Options file you wish to use. If the default checkbox is selected next to the file, that row doesn't get selected and the file from the row previously selected gets loaded. From a user perspective, they think the one they selected with the check box will get loaded. Please also select the row of the file when the checkbox is selected.
