Home page: The Assistants

The Quantix Examinator home page is the page you see when you launch the application. It includes  links to all major actions you will want to do. You can go back to this page at any time by simply clicking the home page tab:

Assistant Description

Analyze one file: Click this link or its image to start the assistant to analyze a single data file.
Use this link when you want to analyze data from a single file. No matter how big your file is, the Examinator will swallow it in seconds! 
Note: To perform multiple analyses of the same file, there is no need to restart the assistant from the very beginning, simply make your changes using the Data and/or Settings pages before clicking the Build report.

Compare files: Click this link or its image to start the assistant to compare multiple files to each other.
Use this link when you want to compare two or more files to each other. This is typically for correlation analysis when you want to compare results from different testers, different program releases, etc.. This assistant generates report sections that let you quickly identify anomalies and helps you improve your process. 
Note: To perform multiple analyses of the same files, there is no need to restart the assistant from the very beginning, simply make your changes using the Data and/or Settings pages before clicking the Build report.

Merge files:  Click this link or its image to start the assistant to merge multiple files and analyze the combined data set.
Use this link when you want to combine data from multiple files and process the resulting data as if it were one big file. This is useful when combining multiple lots, and/or when looking for tests that never fail, and conducting 'Skip Test' engineering optimizations of test programs in order to suppress them from your test flow.
Note: To perform multiple analyses of the same files, there is no need to restart the assistant from the very beginning, simply make your changes using the Data and/or Settings pages before clicking the Build report.

Compare groups of files: Click this link or its image to start the assistant to compare groups of files. This assistant provides full flexibility on which data files to merge and which resulting groups to compare. From this assistant, you'll be able to conduct the most complex data analysis on a large amount of data in just few mouse clicks!
This feature can be used in situations such as comparing production trends between two programs revisions over a long time period.
Note: To perform multiple analyses of the same files, there is no need to restart the assistant from the very beginning, simply make your changes using the Data and/or Settings pages before clicking the Build report.

Neptus: Click on Neptus for detailed hints and precise solutions to help you get the most out of the Quantix Examinator! It also includes all links to the on-line help pages.

Scripting center: Click this icon or text link to load the scripting center page, and unleash the full power of Quantix Examinator.  You can use scripts to fully automate data analysis and report generation, and to send results to other applications for real-time SPC processes. For more details about the scripting language, check the help section.
