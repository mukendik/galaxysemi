Compare Datasets

To compare datasets, launch the Compare Datasets assistant and repeat the following sequence for each dataset to compare:

  • 1: Click the create query icon

  • 2: Fill the query fields. For more details, see single query page.

  • 3: Go to next assistant page to define what to include in your report: charts types, and which specific tests / parameters to plot (settings). Default report includes test statistics tables, Soft bin wafermap, and histogram of all tests.

Load/Save your File List

Examinator allows you to save your list of queries to disk so you can later reload it with just one mouse click! This is a very convenient feature that boosts your productivity on repetitive tasks involving the same queries.

: Click this icon to save your current list of queries to disk (saves a script file)
: Click this icon to reload a list of queries from disk (reads a script file)
