Comprehensive data analysis in just few clicks!

Whatever data analysis task you need to accomplish...its only few clicks away!

Step# Description
1 Launch the Quantix examinator : gex
2 From the home page, click the action you want to accomplish:
 Analyze one file  
 Compare files
 Merge files
 Compare groups of files
This makes the Examinator display the Data page ('Data' tab)
3 Select all the files you want to analyze and/or compare (using the icon to pick files, or to import all files matching a time period). If comparing groups of files you will also have to first create your groups using the relevant icons.
4 To filter some data from a file (such as only using good parts, site #2, etc.), simply double click the file name in the list; this will launch the file property window ()
5 Click the Next button...to review/edit the current default report sections to be generated (e.g. create a specific wafer map, or advanced reports,...), then click the Build report
