Script icon

You may never need to write a single script while using Quantix Examinator (Pro edition only), but you should be aware of all the power, and potential, behind it. Quantix Examinator is totally built around a scripting engine, when you run an assistant or edit your Options, all these actions are scripts to the Examinator! This means that you can smoothly integrate the Examinator into an automated process with very little effort. The Examinator has been built around a scripting engine to ensure that you do not have to compromise: Quantix Examinator will let you create the most appropriate solutions allowing you to leverage all the scripting functions.

When you click the script icon from the home page or the Examinator toolbar, you jump to the scripting center page. This page is split in two zones: the left half includes a list of favorite scripts, and the right half includes the scripting console.

Favorite scripts: When first installed, the Examinator loads this list with some ready-to-use script programs delivered with the Examinator solution. These scripts are provided for your convenience and you can change them as you wish to better suit your needs. Some of these scripts show how to use message/prompt script functions, others focus on the data analysis, while others like games are for your own amusement!
Here are the icons you'll use to operate/modify your list of favorite scripts:

Icon Description

Add a script to your Favorite list. After clicking this icon, browse your disk to select the script file to add to the list, and then click the ok button. Then you'll be prompted to enter the script description that will also appear in the list.

Edit the script description of the selected (highlighted) list entry.

Edit the selected script.

Execute the selected script 

Remove a script from your list

This stop icon is located in the Examinator toolbar at the top line of the application (above the navigation tabs). Click on it at any time to abort a script.

Note: The Favorite scripts list is saved in the file .galaxy_examinator.csl when Quantix Examinator closes, and is loaded at application startup time. This file is located in your windows directory under PC-Windows, or your $HOME directory under Unix.

Scripts window console: This is where all script messages and script prompts are redirected.
Here are the icons used to operate/modify your list of favorite scripts:

Icon Description

Create a script. Launches the text editor so you can start writing a script from scratch. You can save some time by editing an existing one and save it under a different name; this will save you few minutes!

Select a script, and run it immediately. Convenient when creating a new script and debugging it without having it in your favorite list yet.

Edit the script last executed (regardless of whether it was selected from the favorite list, or selected from the icon just above).

Start the last executed script again (regardless of whether it was selected from the favorite list, or selected from the icon just above).

Note: When a script writes a message it is redirected to the script console. This console supports a scroll bar so you can review an important history list of the messages displayed. When the script is prompting the user to input a string, an edit box will appear at the very bottom of the console area. Once the text is entered, either hit the return key, or click the Ok button located at the right-hand side of the text input window. If you wish to enter a complete file name and path, click the browsing button that precedes the Ok button: it will open a browser interface and ease the selection process!
