strIsNumber(
  const string)  // string to test

Returns true if the string represents a number, otherwise false.

Examples:

isNumber = strIsNumber(1.56);   // isNumber = 1
isNumber = strIsNumber('abc');  // isNumber = 0
isNumber = strIsNumber(-123);   // isNumber = 1
