 Overview
This library provides a complete and powerful set of functions to manipulate strings.


Functions

Function Description
strAscii Gives the ASCII value of a character
strBuildRecord Creates character delimited string for export (e.g. for Excel)
strCenter Centers text in a string
strChange Character substitution in string
strChar Gives the character of an ASCII code
strExport Converts special characters to escape sequence (C like)
strFormatNumber Adjusts number in string
strImport Converts escape sequence to special characters
strIndexOf Finds sub-string position in string
strIsInteger Tells if string is an integer number
strIsNumber Tells if string is a number
strLastIndexOf Finds sub-string position in string (starts from end of string)
strLeftJustify Left justifies a string
strLength String length
strLower Converts string to lower case
strParseRecord Reads character delimited string (e.g. .CSV file generated from Excel)
strRemoveWords Removes a word from a string
strRightJustify Right justifies a string
strSplitConnectString Splits a database login string into 4 elements 
strSplitPath Splits path into drive,path,name,extension
strSpread Enlarges string (adding one space after each character)
strStrip Strips characters
strStripExtension Strips file name extension
strStripLeading Strips leading characters
strStripTrailing Strips trailing characters
strSubString Extracts sub-string
strUpper Convert string to upper case
strWordCount Tell number of words in string
strWords Extract words from string
