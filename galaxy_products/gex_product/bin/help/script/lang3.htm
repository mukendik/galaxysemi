Numbers may be express in decimal, octal or hexadecimal notation. While decimal notation can be used for
 integers and floating points, octal and hexadecimal notation allows integers only.

Numbers starting with 0x or 0X are considered hexadecimal, numbers starting with 0 are assumed to be octal and numbers starting with 1 to 9 are interpreted as decimal.

Floating point constants may be written either in fixed point or exponential notation.

Examples:

123        // decimal integer
123.456    // decimal floating point
1.23456e+2 // same value in exponential notation
0177       // octal value
0x1f7      // hex value
