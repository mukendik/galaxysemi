gexInteractive(
  const section,      // section entry name
  const field,        // field name
  const value)        // field valu

Lets you build single layer or multi-layers interactive charts within a script file, then have the Examinator application directly display the resulting charts. This scripting function is only available under Examinator-Pro. Examinator and all Standard Editions do not support this advanced function. With this function, you can:

  • Create interactive Histogram chart (unlimited layers)
  • Create interactive Trend chart (unlimited layers)
  • Create interactive Scatter plot (unlimited layers)

Note: Fields are not case sensitive.


Where to insert these script lines?

As these scripting functions are immediate commands and trigger the Examinator interface to switch to the Interactive page, they should always appear just after the call to the BuildReport() function call; or just after the scripting command gexBuildReport(....) which is located into the function BuildReport()

Example: Code in the script 'main()' function...

....

// Analyze data and build the report, do not show report page
BuildReport();

// Interactive commands start here!
// Remove any existing layer in the stack: start from a blank page!
gexInteractive('reset');

// Create Histogram of test 1060
gexInteractive('setX','1060');

// Insert Layer#1
gexInteractive('addLayer');

// Draw Histogram chart for all the layers
gexInteractive('Chart','Histogram');

 

Here is the list of Interactive actions you can conduct using this script function:

action Description
reset Erases all existing chart layers in memory
setX Defines the Test / Parameter to plot if Histogram or Trend chart (or plot on the X axis if Scatter plot)
setY Defines the Test / Parameter to plot on the Y axis (ignored under Histogram and Trend charts)
addLayer Insert layer previously defined using the setX and setY actions
Chart Triggers Examinator to display the Interactive chart page and show your layers plotted.

Section: reset

Example:

// Remove any existing layer in the stack: start from a blank page!
gexInteractive('reset');

 

Section: setX

Value Description
test_number Where test_number is the test number you want to chart (if charting a histogram or Trend) or chart on the X axis if the chart you build is a scatter plot.
The test number can be in the format XXX (where XXX is the test number, eg: 1060), or XXX.YYYY (where XXX is the test number and YYYY the pinmap index, eg: 1060.17)

Example: Overlay two tests (Interactive trend chart)

// Remove any exiting layer
gexInteractive('reset');

// Define Parameter ID to plot
gexInteractive('setX','1060');
// Insert Layer#1
gexInteractive('addLayer');

// Define Parameter ID to plot
gexInteractive('setX','2078');
// Insert Layer#2
gexInteractive('addLayer');
 // Display Interactive Trend chart
gexInteractive('Chart','Trend');


Section: setY

Value Description
test_number Where test_number is the test number you want to chart on the Y axis for a Scatter plot. This function (setY) is ignored if charting Histogram or Trend charts.
The test number can be in the format XXX (where XXX is the test number, eg: 1060), or XXX.YYYY (where XXX is the test number and YYYY the pinmap index, eg: 1060.17)
 

Example: Interactive Scatter plot (2 layers)

// Remove any exiting layer
gexInteractive('reset');

// Define Parameter ID to plot in X
gexInteractive('setX','1060');
// Define Parameter ID to plot in Y
gexInteractive('setY','1070');
// Insert Layer#1
gexInteractive('addLayer');
      
// Define Parameter ID to plot in X
gexInteractive('setX','1081');
// Define Parameter ID to plot in Y
gexInteractive('setY','1082');
// Insert Layer#2
gexInteractive('addLayer');
      
// Display Interactive scatter plot
gexInteractive('Chart','Scatter');

 

Section: addLayer

Have Examinator internally fills one layer structure (previously defined using setX and setY command). No argument for this command.

Example:

// Insert layer
gexInteractive('add_Layer');

 

Section: Chart

Value Description
chart_type Where chart_type is the type of chart you want to display. It is one of the following charting types: histogram, trend, or scatter. This command has an immediate effect and triggers Examinator to display the Interactive Chart page showing your layers.

Example:

// Draw Histogram chart for all the layers
gexInteractive('Chart','Histogram');
