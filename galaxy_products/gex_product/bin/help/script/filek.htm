fileFindFirst(
  const folder,  // Folder path to scan
  const ext)     // File extensions to look for

Finds all files in the given folder that match the extensions. This instruction must be called on the first loop only (in order to initialize the list of files present in the folder). After the first file name is retrieved using the fileFindFirst command, use fileFindNext to retrieve the following file names.

Example:

var vPath = '/usr/myfolder';
vExt = '*.csv;*.txt';
vFile = fileFindFirst(vPath,vExt);

while(vFile != '')
{
  // Do some work on the file

  // Get next file to process
  vFile = fileFindNext(); 
  };
}
