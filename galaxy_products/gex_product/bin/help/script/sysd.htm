sysPrompt(
  [message,   // message to display as prompt
   raw])      // raw logging mode if true

Displays a prompt and waits for user input. Prompt and entered information are saved to the logfile if one is
open.

Returns the user input.

Example:

var x = sysPrompt('please enter your name: ');
sysLog('welcome '+x);
