 Overview
This library includes all math functions you may need to incorporate complex algorithms in your scripts.

 

Functions

Function Description
abs Computes absolute value
acos Computes arc cosine
asin Computes arc sine
atan Computes arc tangent
ceil Returns smallest integer greater or equal to parameter 
cos Computes cosine
cosh Computes hyperbolic cosine
exp Computes exponential of parameter
floor Computes largest integer smaller or equal to parameter
log Computes natural logarithm
log10 Computes base 10 logarithm
max Returns the MAX of 2 values 
min Returns the MIN of 2 values 
pow Computes X to Power Y
sqrt Computes square root
sin Computes sine
sinh Computes hyperbolic sine
tan Computes tangent
tanh Computes hyperbolic tangent
