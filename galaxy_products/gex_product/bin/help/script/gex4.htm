gexGroup(
  const action,       // action on Group
  const value)        // field value

Lets you manage groups of data files to process. You can erase all current groups structures, and/or
create new empty groups. 
When analyzing one data file or merging multiple files, only one group is needed. When performing
correlation over multiple files, as many groups as clusters of data to compare must be created.
Note: Fields are not case sensitive.

Example:

// Reset all groups
gexGroup('reset','all');
// Because parameters are not case sensitive, it is the same as: 
gexGroup('Reset','ALL');

Here is the list of actions you can conduct on a group using this script function:

action Description
reset Erases all existing groups if any
insert Insert a new empty group in your list

Section: reset

Value Description
all Resets all groups: erases all existing groups and list of files in groups. This function
should be called prior to any other calls to gexGroup and gexFile

Example:

// Erases all groups/file lists structures.
gexGroup('reset','all');

 

Section: insert

Value Description
group_name Where group_name is the name of the empty group to create. Function returns
a groupID to be used for inserting files into this group (using gexFile). GroupID is
set to -1 if an error occurred during group creation.

Example:

// Creates 2 groups for correlation analysis
gexGroup('reset','all');
t1 = gexGroup('insert','Group1: files from tester T1');
t2 = gexGroup('insert','Group2: files from tester T2');
