 Overview

Gex Report Library functions let you fully control the reports you generate. You can use any of these functions without any prior declaration in your script. Because all libraries listed in the manual are embedded in the Examinator, you just use them as-is!
Commonly used functions include:

  • Define the files to process and compare
  • Define all the options (report type, fields to display, charts sizes, etc.)
  • Define the report pages to create (wafer map, trend chart, etc.)
  • Define which tests and parts to consider
  • and much more!

Looking for a shortcut? Use the ready-to-use scripts to see how easy it is to control the GEX application!

Functions

Function Description
gexBuildReport Analyze data files, and generate report
gexFavoriteScripts Set the list of 'Favorite Scripts' in the GEX Scripting center page
gexFile Add a file to a group
gexFileInfo Returns information about a  file (eg: LotID, Tester name, etc...)
gexGroup Create/remove groups of files
gexInteractive Allows to build and display interactive charts
gexOptions Set ALL the GEX report Options!
gexParameter Customize parameters Test Limits, add your own Markers, etc...
gexReportType Define the type of report to generate (tables, charts, wafer map...)
gexPromptQuery Prompts user to pick valid values for a given database key (eg: LotIDs...)
gexToolbox Automate some of the Examinator toolbox functions
