while (expression) statement

While expression evaluates true statement is executed. (The check is done before statement is
executed).

The loop may be left premature by break. With continue the rest of statement is skipped and the loop
continues with the next expression check.

x=1;
y=1;
// Compute y = 1*2*3*...*7
while (x <= 7)
{
  y = y*x;
  x = x+1;
}
