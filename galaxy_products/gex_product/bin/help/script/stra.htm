strIsInteger(
  const string)  // string to test

Returns true if the string represents an integer (long), otherwise false.

Example:

isInteger = strIsInteger(1.56);   // isInteger = 0
isInteger = strIsInteger('abc');  // isInteger = 0
isInteger = strIsInteger(-123);   // isInteger = 1
