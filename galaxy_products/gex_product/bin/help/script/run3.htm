 CSL first saves all command line parameters into a global array named mainArgVals. Then it loads
 the script given as parameter 1 and calls the function named main.

Check it out by writing a new program args.csl:

// args.csl
// Main entry of the Script...
main()
{
   for (var i=0; i<sizeof(mainArgVals); i++)
      sysLog(mainArgVals[i]);
}

Run this sample with the command:

gex args my name is Quantix

The output will look like:

gex
args
my
name
is
Quantix
