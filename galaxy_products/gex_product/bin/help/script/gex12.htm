gexPromptQuery(
  field,	// Database field to query (eg: LotID)
  message,	// message to display as prompt
  [ raw])	// raw logging mode if true

Displays a prompt and waits for user input. Prompt and entered information are saved to the logfile if one is
open. Also, if the user clicks the browse icon to the right of the edit box, a dialog box appears listing all the valid values matching the key filter criteria (eg: all Lots in a the database, or all testers, etc).
Notes:

  • The definition of the database name, the time period and filters must be defined first prior to call this function (see below example).
  • The database name must be the database logical name.
  • The database type must be left empty for all file-based databases (non Quantix SQL databases)

This function return the user input or selection made thru the browse dialog box.

If database accessed is SQL database, then the field specified is the same field as available under Examinator user interface (eg: 'Lot ID', etc; including support for meta-data fields). In all other cases (including SQL database accessed in off-line mode which consist in reading data from local cache folder), you must use the filter names 'dbf_x' as generated by Examinator in script function 'gexQuery' when creating queries. E.g: 'dbf_product_id' filters on the ProductID; 'dbf_lot_id' filters on LotID, etc...To know the exact filter names to use, best is to run Examinator and run a query with the filters you want to define, then run the report. Upon report creation, go to the scripting page and edit the script just executed to see the script lines starting like:  gexQuery('dbf_'

 

Example (for non-SQL database):

// Define database to query
gexQuery('db_name','Db_MEMS_PCM');
// Ensure we disable the cache
gexQuery('db_offline_query','false');
// Database type (ignored for non SQL databases)
gexQuery('db_data_type','Final Test data');
// Define 'all dates'
gexQuery('db_period','all_dates','2007 5 30 2007 5 30');

// Prompt user to enter a LotID or click the 'browse' button.
var x = gexPromptQuery('dbf_lot_id','please enter lots: ');
sysLog('You've selected the following lots: '+x);
