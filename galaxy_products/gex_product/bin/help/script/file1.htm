File library
     

 Overview
This library includes all functions needed to create, read, and write files (it supports both ASCII and Binary
formats). This makes your script open to the outside world and let you read configuration file, test lists
and much more...


Functions

Function Description
fileClose Close a file
fileCopy Copy a file
fileDelete Delete a file
fileDelDir Remove a directory
fileEof Check for end-of-file
fileExists Tells if files exists or not
fileFindFirst Enumerate files found in a given folder (first call only)
fileFindNext Enumerate files found in a given folder (second and following calls only)
fileFlush Flush file output buffer
fileInfo Get file info (size, date, etc.)
fileLocate Find file in given paths
fileMakeDir Create a directory
fileOpen Open a file
fileRead Read characters from a file
fileReadLine Read one line from a file
fileReadPos Get the file-read position
fileRename Rename a file
fileTempName Get a unique temporary file name
fileWrite Write characters to a file
fileWriteLine Write one line to a file
fileWritePos Get the file-write position
