 C Scripting Language (CSL) is very close to C. However there are differences; the most significant are:
  • The one and only variable type is var which (similiar to REXX variables) holds any number or string. No pointers, struct's and typedef's. (struct functionality is however possible with arrays) 
  • No goto's.
  • Exception handling by try/catch/throw.
  • Dynamic array allocation and reallocation.
