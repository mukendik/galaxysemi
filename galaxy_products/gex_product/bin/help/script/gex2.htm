gexBuildReport(
	[const show_page])  // page to show

Starts data analysis and generates the report. If show_page  is omitted, the report home page is shown. 

For a complete example, take a look at the ready-to-use template wafermap.csl

show_page Description
hide or
none		 No page displayed, Examinator remains on the default page that launched the script.
home Launch report on its home page
global Launch report on the Global information page
statistics Launch report on the Test statistics page
histogram Launch report on the Histogram page
wafer Launch report on the Wafer map page
binning Launch report on the Binning page
pareto Launch report on the Pareto page
advanced Launch report on the Advanced report page

 

Example:

// Calls user function to set options
SetReportOptions();
// Calls user function to set report type to generate
SetReportType();
// Call GEX function to analyze data+create report
gexBuildReport('home');
sysLog('Report Generated!');
