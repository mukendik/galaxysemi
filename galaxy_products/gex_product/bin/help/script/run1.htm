 The GEX Quantix Examinator executive program comes with built-in scripting support. It allows you
to leverage the power of scripting language by writing custom scripts that do exactly what you want!

Syntax to launch the Examinator and have it run a script:

gex [-options] [scriptfile] [additional parameters]

The scriptfile extension may be omitted if it is .csl, the additional parameters depend on your script file.
The -options parameter  lets you provide run time options to the Examinator, and lets you decide how
the Examinator must behave while processing the script (e.g. run minimized in icon, close once script is 
completed, etc...). Check the Runtime options page for a full description of the options available.

The following script example creates a gradient waferwap for parametric test# 1020, and executes with GEX hidden, and has GEX close after the script has been executed: 

gex -HCS /export/home/myscripts/wafer_map.csl 1020
