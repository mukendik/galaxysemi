for ([expression-1]; [expression-2]; [expression-3]) statement

expression-1 is executed once to initialize the for statement. As long as expression-2 evaluates true
statement and expression-3 are executed in sequence. If expression-2 is not present, it is taken as
permanently true.

The loop may be left premature by break. With continue the rest of statement is skipped and the loop
continues with the next expression-3 followed by the expression-2 check.

Example:
y=1;
// Compute y = 1*2*3*...*7
for(x=1;x<8;x=x+1
{
  y = y*x;
}
