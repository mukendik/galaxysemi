sysHtml(
  const action,      	// action on the script console HTML window
  [const path])    	// path parameter if needed

Lets you show/hide the HTML window located at the right side of the script console. Also, lets you load and display a specific HTML page into it. At launch time, the Quantix Examinator hides the HTML window, so to display pages, you ALWAYS need to call this function with the 'show' action.

Note to Windows 95/98 users: you may experience compatibility issues when using this command as the latest Microsoft patches are required.

This command lets you leverage the power of the HTML language and insert images/links to bring a more professional style to your scripts!

Note: Loading a page does not force the HTML window to be visible if it is currently hidden.

Example:

// Hide the HTML console window
sysHtml('hide');
// Show the HTML console
sysHtml('show');
// Load a HTML page
sysHtml('load','c:\\temp\\index.htm');

Here is the list of actions you can conduct on a file using this script function:

action Description
hide Hides the HTML console window
show Shows the HTML console window
load Lets you load a specific HTML page into the HTML window 
 

Section: hide

Fields Description
path Ignored, and not needed

In this mode, the HTML window is hidden and the Examinator resizes the script console window to use all the freed space.

Section: show

Fields Description
path Ignored, and not needed

In this mode, the HTML window is displayed and the Examinator resizes the script console window to a smaller size so both windows can be seen side by side.

 

Section: load

Fields Description
path The full path to the HTML page to load

This function loads the HTML page into the HTML window, but is only visible if you made the HTML window visible using the command line sysHtml('show').

Example:

// Load a specific HTML page
sysHtml('load','/export/home/demo/hangman1.htm');
// Now show it!
sysHtml('show');
