fileCopy(
  const source,       // source file name
  const dest)         // destination file name

Copies a file.

This is a binary copy ignoring any system attributes. If you'd like system attributes to be copied use sysCommand('copy ....') instead.

Example:

fileCopy('file1.txt', 'file1.txt.tmp');
