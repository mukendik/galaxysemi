fileInfo(
  const filename)  // name of file queried

Returns detailed information about a file:

  • The size of the file in bytes
  • The last access date/time as YYYYMMDDHHMMSS
  • The last modification date/time
  • The creation date/time
  • The full filename

You can extract the parts with strWords, but keep in mind there may be filenames containing spaces (if
running under Windows).

Example:

fileInfo('test.txt');
// '125 19980205142536 19980205122807 19980205121355 G:\Quantix\Examinator\test.txt'
