switch (expression-0)
{
  case expression-1: [statements]
  [case expression-2: [statements]]
  ...
  [case expression-N: [statements]]
  [default: statements]
}

expression-0 is compared with expression-1 ... expression-N. As soon as a match is found, all
subsequent statements within the switch block are executed. The statements after default are executed if
none of the previous expressions were matched.

Within the statements, break is used to leave the switch block premature.

Note that unlike C, expression-1 ... expression-N don't have to be constants.

Example:

switch(MonthOfBirth)
{
   case 1: sysLog("You're from January!");
      break;   
   case 2: sysLog("You're from February!");
      break;
   case 3: sysLog("You're from March!");
      break;   
   case 4: sysLog("You're from April!");
      break;
   case 5: sysLog("You're from May!");
      break;   
   case 6: sysLog("You're from June!");
      break;
   default: sysLog("You're from the second half of the year!");
      SpecialMessage();	// Display extra message...
      break;   
}
