 Opens or closes a logfile at compile time. Useful when generating P-Code listings (see #list). This directive
uses the system library so it has to be loaded in advance:
// CSL Script first line...
 
#logFile 'listing.lst'    // open logfile
 
....
 
#logFile ''               // close logfile

If no extension is given, CSL will add the default extension .log.
