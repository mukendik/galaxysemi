strCenter(
  const string,   // string to center
  const width,    // width of field
  [const padd])   // padding character (default = ' ')

Centers a string in a field of width.

Examples:

y = strCenter('hello', 11);        // y = '   hello   '
z = strCenter('hello', 11, '*');   // z = '***hello***'
