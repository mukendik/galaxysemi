Script: args.csl

This script shows how to retrieve the arguments passed when calling the script. 
When launching the Quantix Examinator with a script name, any string following the
script name are the arguments (arguments are separated by spaces).

To the command line: gex args.csl  x1  x2  x3
The program will display 3 arguments: x1, x2 and x3

Additional information can be found in the scripting language documentation, and reviewing the other sample scripts.

The source code follows:

//////////////////////////////////////////
// Welcome to Quantix Examinator Script
// Find more on www.mentor.com

//////////////////////////////////////////

main()
{
   for (var i=0; i<sizeof(mainArgVals); i++)
      sysLog(mainArgVals[i]);
}
