sysLogFile(
  [const filename]) // name of logfile to open

Opens or closes a logfile. If no argument is given, the currently open logfile is closed. If an argument is given,
the currently open logfile is closed and a new one is opened. The logfile is always opened in append mode.

If no extension is given, .log is appended.

Returns name of logfile.

Example:

 
main()
{
  // open logfile with same name as script but extension .log
  sysLogFile(strStripExtension(cslFileName));
  ...
}
