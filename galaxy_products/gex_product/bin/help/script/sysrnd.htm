sysRandom(
  const maximum)  // Random Maximum integer value

Returns a random positive integer value between 0 and maximum.

Example:

const random = sysRandom(60);  // Returns a random number between [0-60]

Note: If you want to generate floating point random numbers, you need
to divide the integer number by 10, 100, or 1000 depending how many digits
you need.
