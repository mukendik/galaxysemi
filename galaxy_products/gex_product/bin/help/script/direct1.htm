 Directives are introduced by the # character. Unlike C and C++ directives are not processed by a pre-compiler but by the compiler itself. They serve as commands executed at compile time.

One effect of this is that you may use const's and constant expressions as arguments for the directives.

Directives include:

Directive Comment
#include Includes a script file at execution time
#list Enables/disables P-Code listing
#loadScript Compiles another csl source file at compile time.
#logFile Opens or closes a logfile at compile time
#if / #else / #endif Enables conditional compilation
#message Prints a message at compilation time
#error Forces an error at compilation time
#block Names a code block in order to trace it
