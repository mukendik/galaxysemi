fileLocate(
  const filename,  // name of file to search for
  const varname)   // environment variable with path's

Searches for a file in all directories listed in a specific environment variable such as PATH, INCLUDE, LIB and so on. Returns a fully qualified pathname of the found file. If the file is not found an empty string is returned.

Example:

var webnavigator = fileLocate('netscape', 'path');
if (webnavigator == '') throw 'netscape not found';
// Shows 'netscape = <path+name>'
sysLog('netscape = '|webnavigator );
