Scripting Samples

Here is a great collection of scripts sample source codes you'll find:

  • Generic scripts help you start with very basic examples

  • Setup scripts show you how to fully manage the GEX setup

  • Data analysis scripts show you real case studies

  • Interactive scripts show you that you can interact with scripts to make them even more flexible

  • ** GAMES ** !!!...yes, we couldn't resist creating few classic games!

Review them to learn more about the language and to save valuable time when creating your own scripts!
You'll find much more Scripts and papers on the Quantix web site, visiting the GEX Freeware Center

Generic Scripts Description
hello.csl Prints: Hello Quantix World!
trace.csl Shows how to use Trace function
args.csl Shows how to use command line arguments
sys_show.csl Shows Examinator Script engine internal information 
Examinator Setup Description
options.csl Shows how to configure the report you want (charts, tables, etc.)
favorites.csl Have Examinator load your own list of Favorite Scripts
Data analysis Scripts Description
statistics.csl Generate the list of tests with Cpk<= 1.33 : report launched on the Test statistics page
wafermap.csl Generate SOFT bin wafer map: report launched on the wafer map page
wafermap_gradient.csl Generate wafer map of a specific parametric test
histogram.csl Generate standard histogram over test limits
adv_histogram.csl Generate advanced histogram over test results
adv_trend.csl Generate trend chart over test results
adv_correlation.csl Generate correlation (XY-Scatter) chart between 2 tests you select
adv_boxplot.csl Generate Gage R&R - box-plot chart of all tests
adv_datalog.csl Generate datalog report of tests 1000 to 1005
Interactive Scripts Description
menu_wafermap.csl Shows a menu to let you select the wafer map type (soft bin, hard bin, parametric),
 and the test number (if parametric wafer map).
menu_datalog.csl Shows a menu to let you define the datalog options (all tests, failing tests, or specific tests).
Interactive Games! Description
hangman.csl Remember the hangman game from when you were a kid?! You can again enjoy this classic!
hangman-html.csl A much sexier hangman--it uses HTML pages instead of text messages!
   

Note: These scripts are part of the Quantix Examinator software. You will find them in the GEX_PATH/samples directory, where GEX_PATH is the path where you have installed the Quantix Examinator application.
