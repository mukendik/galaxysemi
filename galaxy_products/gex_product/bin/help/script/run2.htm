 Start out writing a small test program with your favorite text editor and save it to a file named hello.csl:
// HELLO.CSL Script
 
main()
{
  sysLog("hello Quantix world");
}

Now run it by entering this command line:

gex hello

Or run it from the Examinator interface by clicking the toolbar Script icon:
Then add your script to the favorite list and run it, icons: and

NOTE: You can find all the samples shown in this manual in the Samples subdirectory.
However, you might find it better to manually enter the smaller samples because 
you will learn more through the inevitable typo errors in your script.
