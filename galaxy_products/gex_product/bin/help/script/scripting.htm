Which GEX releases support scripting?

GEX release Supports scripting
GEX Evaluation (10 days) Yes. During the 10 days trial period, you can use all the scripting related functions and even create/run your own custom scripts.
GEX Server Yes. Any GEX client using a floating license can run and create its own scripts. This is the best solution if you are looking for a high level of interoperability with other software and look for automated processes.
GEX Standalone No. Custom scripts can't be created, but all GEX standard functions remain available. The GEX standalone is the best release for users looking for a turn key ready-to-use solution.

Some background

The Quantix Examinator C Scripting Language (CSL) is a well structured and easy to learn script  programming language. CSL follows the C syntax very closely and programmers accustomed to C, C++ and JAVA will immediately become familiar with it. CSL is used like an interpreter: You write the program with your favorite editor and run it directly from the Examinator interface, or calling the examinator with the shell program path and name in the argument).

A powerful set of included libraries covers topics like system, strings, math, files, and Examinator.

Using CSL, you leverage the Examinator power by seamlessly interfacing to any other software or process you want!
