 String literals may be enclosed in single or double quotes: 
'This is a string'
"This is also a string!"

Several consecutive literals are concatenated to a single string:

"john's" ' car'

evaluates to exactly the same as

"john's car"

Literals must terminate before the end of a line, consequently you must use the concatenation feature for long literals continue over several lines.

Encoding Nonprintable Chars

Nonprintable characters are noted with a backslash followed by an encoding character or number:

 
Written Description
\e escape (ESC)
\t horizontal tab (HT)
\r carriage return (CR)
\n newline/line feed (LF)
\f form feed (FF)
\\ backslash (\)
\' single quote (')
\" double quote (")
\oOO octal character code (O standing for octal digits 0...7). E.g..: \o77
\xhh hex character code (h standing for digits 0...9 and a..f). E.g.: \x7c
