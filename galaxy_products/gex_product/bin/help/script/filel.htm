fileFindNext()

Finds all files in the given folder that match the extensions. function to be called after the first file name has been retrieved with the fileFindFirst command

Example:

var vPath = '/usr/myfolder';
vExt = '*.csv;*.txt';
vFile = fileFindfirst(vPath,vExt);

while(vFile != '')
{
  // Do some work on the file

  // Get next file to process
  vFile = fileFindNext(); 
  };
}
