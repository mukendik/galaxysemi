Includes a file at compile time.

#include 'waferlib'

...or...

static const waferLib = 'waferlib';

#include waferLib

If no extension is given, CSL will add the default extension .csl. The file is searched in the paths defined in
environment variable CSLPATH. If no CSLPATH is defined the file is searched in the current working
directory.

NOTE: #include will perform unconditionally which may by a problem when the same file is included from several sources. You might be better off using #loadScript to load forward declaration because it will automatically avoid multiple loads (as you would do in C/C++ by a sequence of #ifndef _XXY_ ... #define _XXY_ ... declarations ... #endif)
