To select your dataset:

  1. Enter a Query name
     

  2. Database source
     

  3. Time period
     

  4. Select Filters
     

  5. Click Next button

Advanced help
More details:

Query name: This is the title of your Query (e.g: Correlation on TX3), you'll see it in all the reports

Select Database source:  Selects the database (data collection source) on which the query needs to be executed.

Time period:  Select the time period filter (if applicable) ; data with a timestamp outside of this period are ignored. Select All dates if filtering your data on time & date is not relevant.

Filters:  Select the appropriate filters you want (not case sensitive), wild characters allowed ( * and ?), and to combine multiple values for a same filter, use the '|' character (without any space).

E.g: To have the LotID filter combine all data from lot1 and lot2, enter the string: Lot1|Lot2

Next: Same as clicking the Settings tab. It brings you to the next assistant page to select the report sections to create and/or the interactive drill to perform.
