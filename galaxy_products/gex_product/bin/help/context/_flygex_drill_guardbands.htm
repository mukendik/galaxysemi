Advanced help

More details:

Double click the test limit to edit then enter the new value. Press the Return key to validate your changes; the cell should change to a yellow background. To set multiple consecutive limits to the same value, simply select the first cell then drag the mouse down to the last cell to edit, type the new value and press the Return key. To reset all changes, click the Reset button.

 

Check the 'Reflect changes in all reports' box if you want all the Examinator report pages or files show and use the new test limits entered from this page. Then all pages, including test statistics information will be updated accordingly.

 

Press the Compute button to have GEX compute the new test statistics (using your new limits). Affected values appear in green. To reset all changes, click the Reset button

 

: After playing with the test limits, and when reaching results you want to save, simply click the snapshot camera icon to update your HTML Snapshot Gallery. Only tests edited are saved into the report.
