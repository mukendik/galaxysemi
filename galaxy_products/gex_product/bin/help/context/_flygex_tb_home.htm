Advanced help

STDF Converter
You'll click this link when you want to convert ANY semiconductor data files into STDF V4. Examinator ToolBox allows to convert many different files such as Foundry files (TSMC .WAT, UMC .PCM), Test files (ATDF, GDF) or Excel/CSV files.

Excel/CSV Converter
You'll click this link when you want to convert ANY semiconductor data files into Excel/CSV ASCII. Examinator ToolBox allows to convert many different files such as Foundry files (TSMC .WAT, UMC .PCM), Test files (ATDF, GDF) or STDF V3/V4 files.

Data Editor
You'll click this link when you want to edit/modify ANY semiconductor data file. Examinator ToolBox lets you edit many different files such as Foundry files (TSMC .WAT, UMC .PCM), Test files (ATDF, GDF) STDF V3/V4 files or Excel/CSV then save changes into a Excel/CSV file.

Context help
You can launch the 'Me' (Neptus contextual help) at any time clicking the  icon in the application toolbar. You can drag this help to any place you want in the GEX window so as to best organize your working environment. If you want temporarily to hide this context help, you can simply click on 'Me' (Neptus), this will close the HTML contextual help. However, it will remain present so that a single click on 'Me' can start 'Me' up again. You can completely hide 'Me'  if you want: simply right click over 'Me' then select the hide menu item.
