Advanced help

Master the Interface

  Object Info: Select this icon,  then click any object to display its information. It will give you a sharp and detailed description of your selection (e.g.: die location, binning, failing test, test statistics, etc.). Or click the background to have global information about the file processed, yield data, etc...

Zoom: Click this icon to enter in zoom mode. Then Left click on any object to zoom in on it, or Right click to zoom out. Also CTRL+Left click switches from Zoom to select object mode. This is very convenient as you usually end a zooming sequence by selecting an object and see its information.

Move: Click this icon to enter in Image Move mode. Then click and drag any object of the 3D window to have all the image shifted. Drag to the left to have the image shift to the left, up to shift up, etc. If you keep the Control key (CTRL) down while dragging the mouse, you'll perform a view Rotate instead (very convenient for multiple Rotate/Moves sequences).

Rotate: Click this icon to enter in Image Rotation mode. Then click and drag any object of the 3D window to have all the image rotated. Drag to the left to have the image rotate to the left, up to rotate up, etc. If you keep the Control key (CTRL) down while dragging the mouse, you'll perform a view Move instead (very convenient for multiple Rotate/Moves sequences).

Full Screen: Click this icon to have the 3D window cover all the GEX interface, click on it again to toggle back to standard size within the GEX interface. The Full Screen mode makes navigation more comfortable, it brings sharper details rendering. This selection is also available from the context menu (mouse right click)

Snapshot: Click this icon to take a snapshot of the 3D window, and have it added to your report Snapshot Gallery section.  This selection is also available from the context menu (mouse right click).

Bookmark Home location: Click this icon to have GEX record your current view point (position in space, and zoom angle). Can also be done pressing the B key.  This selection is also available from the context menu (mouse right click).

Go Home: Click this icon to go back to the view point position you book marked with the above bookmark icon. If no previous position was book marked, GEX reloads the original view point and zoom at assistant startup time. Can also be done pressing the H key.  This selection is also available from the context menu (mouse right click).

To make the image Spin, select the Move icon, then click any object in the 3D window and 'throw' it using the mouse (drag the object with the mouse clicked, then release the mouse button while still dragging the mouse). The spinning speed depends on how firmly you 'threw' the object! The spinning axis is always orthogonal to the mouse direction at the time of throw. To stop the spin, click any location in the image.

 

Keyboard Shortcuts

The following table includes all key sequences you can use when the focus is on the3D window. To have the focus correctly set, click the Move icon, then click any pixel of the 3D Window.
Up : Rotate 'back' around X axis
Down : Rotate to 'front' around X axis
Right : Rotate right around Y axis
Left : Rotate left around Y axis
a :  Move to view all the objects on screen (fit to screen)
b : Bookmark current position as HOME location
h : Move to the HOME location
p : Toggle projection mode between Orthographic and Perspective.
+ : Zoom in, magnify the objects without changing your view point origin.
- : Zoom out, shrink the objects without changing your view point origin.
CTRL + Up: Shift image Up
CTRL + Down: Shift image Down
CTRL + Right: Shift image Right
CTRL + Left: Shift image Left
SHIFT + Up: Move forward into the image, this is not a zoom as your viewing location is moved forward (it's like walking toward the objects).
SHIFT + Down: Move backward, this is not a zoom as your viewing location is moved backward (it's like walking away from objects).
