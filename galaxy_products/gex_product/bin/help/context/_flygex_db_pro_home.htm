Analyzing a Dataset
You'll click this link when you want to analyze a specific dataset (query). This is the typical situation during standard engineering, characterization missions, and lot/sub-lot analysis.

Compare Datasets
You'll click this link when you want to compare multiple datasets (queries) with each other. This is the typical situation when you want to perform correlation analysis. 

Database Administration
 Click this link to have access to the Database administration page: create, delete a database, and post all the files you want into the database!

How to...
Click this link and have access to the most valuable and practical data. From here, you'll see how to deal with typical case studies, and save valuable time using the detailed description to make your own similar data analysis.

Scripting
 Click this link to have access to the Scripting page: create, run your own scripts, and maintain your own  list of preferred scripts!

Analyzing one file
You'll click this link when all your data to process are in a single data file. This is the typical situation during standard engineering and characterization missions, or in some cases for single lot/sub-lot analysis.

Comparing files
You'll click this link when you want to compare multiple data files with each other. This is the typical situation when you want to perform correlation analysis. 
E.g.1: tester correlation can be done comparing the same golden parts tested on different testers.
E.g.2: correlation over time can be done comparing the same golden part tested on the same tester at different time intervals.
E.g.3: Compare from the same data file,  test results obtained from different tester sites. 

Merging files
You'll click this link when you want to merge data from multiple data files. This is the typical situation when you want to perform analysis over a long time period such as production trend.

Comparing groups of files
You'll click this link when all other links do not provide enough flexibility! From this link you can create as many groups as you want,  and insert as many data files per group as you want. The data analysis consists of comparing the data from each group. This is the typical situation when you want to compare trend of testers over a long time period.

Context help
You can launch the 'Me' (Neptus contextual help) at any time clicking the  icon in the application toolbar. You can drag this help to any place you want in the ExaminatorDB window so as to best organize your working environment. If you want temporarily to hide this context help, you can simply click on 'Me' (Neptus), this will close the HTML contextual help. However, it will remain present so that a single click on 'Me' can start 'Me' up again. You can completely hide 'Me'  if you want: simply right click over 'Me' then select the hide menu item.  Examinator supports Drag & Drop, so simply drop the files to analyze!
