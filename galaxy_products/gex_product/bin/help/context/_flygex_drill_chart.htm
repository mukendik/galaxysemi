Interactive chart:

  1. Clickto browse your test list or pick a test

  2. Zoom in chart

  3. Switch charting mode Histogram, Trend, Scatter

  4. From the 'Style' tab, define the markers to draw

  5. From the 'Table' tab, see raw test results, and export them to Excel/CSV format if needed

Advanced help
More details:

: Browse you tests/parameters to chart the one you want. Or use the pick list icon to directly pick one specific parameter

: Zoom in (Left click) or Zoom out (tight click) any section of your chart. Also, use the 'samples to plot' combo box for easy quick charting the data over the specification limits or over the data samples. When in histogram mode, you can also switch to a cumulative representation.

: Lets you toggle the charting mode. Data can be charted in Histogram, Trend chart or Scatter plot (in this last mode, you have two sets of navigation arrows to change the test used in the X axis, and the one in the Y axis).

: Lets you save the data samples listed the table into a Excel/CSV file.
