What you can customize:

- Click this tree entry to define the local and server databases paths (Examinator Production only).

- Click this tree entry to customize the content of the HTML pages created automatically on your intranet. These pages include information about products and testers in operation as well as the current yield per product and per lot/sub-lot. This entry also lets you define the purge policy for the history pages (Examinator Monitoring only)

- Click this tree entry to display the report output options you can customize. You can change the output format (HTML or CSV), and define where reports must be created.

- Click this tree entry to display options you can enable/disable that help maximize report creation speed (Not available under Examinator Standard editions)

Define data processing options:
Sorting mode: define if data have to be sorted by date or processed in the order entered.
Failures count mode: define if stop counting on first fell in part or continue counting to the end.
Multi-results parametric tests: define if each pin has a unique test number, of if all pins in a same test merged under a same test name
Outlier filtering policy to activate. You can instruct GEX to process all test results, or select the outlier window (e.g.: 150% of the test limits).

- Click this tree entry to display the test statistics table options. You can set which statistical information to list in the tables, how to sort the information, and for file comparison: define the drift alarm levels (tests over the limit are shown on a red background for HTML reports). If running Examinator Pro, you'll have a selection of advanced statistical information such as Quartiles, Skew and Kurtosis values, etc.

- Click this tree entry to display the wafer map options. You can define the chart size to create, and markers to add on the chart (e.g.: outline dies retested, write bin result in dies).

- Click this tree entry to display the advanced histogram options. This only applies to advanced histograms, not basic histograms. You can define the chart size to create, and markers to add on the chart (e.g.: test name, test limits, 2 sigma space, etc.). 

- Click this tree entry to display the trend chart options. You can define the chart size to create, and markers to add on the chart (e.g.: test name, test limits, 2 sigma space, etc.).

-  Click this tree entry to display the Correlation / Scatter / BiVariate chart options. You can define the chart size to create, and markers to add on the chart (e.g.: test name, test limits, 2 sigma space, etc.).

- Click this tree entry to display the Gage R&R Box-Plot chart options. You can define the chart size to create, and the sorting mode.

- Click this tree entry to display the Binning options. Lets you decide how binning summary is computer (handle or not retests).

- Click this tree entry to display the Datalog options. You can define which information other than test results to include in the datalog report. E.g: test limits, test name, die location, etc.

- Click this tree entry to define the application preferences (eg: text editor, etc).
