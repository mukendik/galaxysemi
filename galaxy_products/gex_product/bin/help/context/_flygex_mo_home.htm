Context help
You can launch the 'Me' (Neptus contextual help) at any time clicking the  icon in the application toolbar. You can drag this help to any place you want in the ExaminatorDB window so as to best organize your working environment. If you want temporarily to hide this context help, you can simply click on 'Me' (Neptus), this will close the HTML contextual help. However, it will remain present so that a single click on 'Me' can start 'Me' up again. You can completely hide 'Me'  if you want: simply right click over 'Me' then select the hide menu item.
