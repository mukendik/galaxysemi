To compare your files:

  1. Clickto browse your disc and select each  file to process (or use Drag & Drop)

  2. From your list, select all the files that need a custom filter

  3. If data processing order is important, sort files in right order

  4. Click to set the parts/binning and sites filters for all files selected
    e.g.: Parts 1-10, 11, 15

  5. Double click any file in your list to change its group name (if needed)

  6. Click Next button

Advanced help

More details:

: Browse you disk or network and select the files to process. The first file you select is considered the reference, and all drifts are computed from it. Multiple selections is supported. Examinator supports Drag & Drop.

 

Files selection:
 Select all files in your list using the mouse and holding the Ctrl key to perform multiple individual select/unselect or the Shift key to select a range of files. 
Hint: The Ctrl+A key sequence allows to select all files at once.

 

Files order:
Examinator default setting is to process files in the same order as listed (unless you change this default to sort by testing date from the Options page). File order can be changed using the arrow icons while selecting a single file from the list.

 

: Lets you define a filter for the data extracted from each individual data file. You can select multiple files to force the same filter at once. Hold the Ctrl key to perform multiple individual selections or the Shift key to select a range of files in your list. The filter accepts that you enter a list of part numbers (e.g.: 1 to 12, 19, 25), or binning. Also allows you to filter on a specific site number. Properties also display file information such as: tester used, date info, operator name, etc (except when multiple files are selected).

Next: Same as clicking the Settings tab, brings you to the next assistant page to select the report sections to create and/or the interactive drill to perform.

: Lets you remove the selected file(s) from your list.

Examinator allows you to save your list of files and filters to disk so you can later reload it with just one mouse click!

: Click this icon to save your current list of files and filters to disk (saves a script file)
: Click this icon to reload a list of files and filters from disk (reads a script file)
