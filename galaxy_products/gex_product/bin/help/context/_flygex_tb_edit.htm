Examinator ToolBox lets you read many different data files, including: Foundry files (TSMC, UMC), Test data (Wafer, Final test), or even Excel/CSV files! As Examinator ToolBox allows unlimited number of columns to be edited, it provides a much more powerful environment than text editors or Microsoft Excel.
