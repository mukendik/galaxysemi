- When you click this icon, a dialog box lets you decide which task type you want to create. This includes:
Datapump
   Yield monitoring
   Spec monitoring
   Reporting
Production status

DataPump: This task allows to setup automatic data import from given folders into specific databases. Allows basic yield monitoring with email alarm notification on all incoming data.
Yield monitoring: This task allows to setup a specific Yield monitoring rule for a given Product.
Spec monitoring: This task allows to setup a specific Parameter information monitoring rule (e.g: check Mean, Cp, or Cpk) for a given Product.
Reporting: This tasks allows to automatically trigger a query and generate its associated report at specific time intervals.
Production status: This tasks allows to have HTML pages created automatically on your intranet to reflect the current production status (testers and products under tests, yield level per product and lot/sublot/tester).
 

- When you click this icon, you can edit/modify the information relative to the task currently selected.

- Makes the selected task to be immediately executed, regardless of its default frequency.

- Destroy all the selected tasks
