Basic reports:

- This report section includes all test statistics information in table format. Typical information listed include: Test number, test name, test limits, mean, range, sigma, Cp, Cpk, etc. You can customize the statistical information included in the report, and the sorting criteria used, from the Options page in the Test statistics section.
Note: When files or groups of files are compared, this table can also include the drift in mean, sigma and Cpk. From the Options page, you can set and activate drift alarm limits so as to highlight, in red, any excessive test drift.



- This report section includes the 2D Wafer maps found in your files. It also lists summary information such as: the start/end date of the wafer map generation, the yield, and the list of bins. If you perform a test parametric wafer map, then each die is not colored based on its binning result, but based on the value of the specific test selected. This representation is very useful to diagnostic topologic process issues. If test limit(s) is missing, Examinator uses the Min/Max values instead. From the Options page, you can customize the wafer map image size, and enable/disable markers such as: outline dies re-tested, or write bin number in the die cell.



- This report section includes the histogram charts for all tests with valid test results found in your file. If your file is created with sampling activated, it may happen that some tests never have any result dataloged. In such a case, no histogram is available, and the test number in the test statistic table is written in black with no hyperlink to the histogram section.
Tip: When comparing files, this report generation may take significant time if all tests are selected (the default)...so disable this report generation if you don't need it! 
Note: This report does not have any options: the histogram section in the Options page is for the advanced histograms only.

Advanced reports:

- This report section gives more flexible histograms charts than in the basic report. It also supports customization from the Options page, section Histograms. You can define the chart size to create, and enable/disable markers to add on the chart such as: test name, mean, limits, 2 sigma space, 3 sigma space or 6 sigma space.
Tip: When comparing files, this report generation may take significant time if many tests are selected...so disable this report generation if you don't need it! 



- This report section creates trend chart (Test value, Difference between tests, Binning/Yield in production). It is the typical information you'll need to see how data behaves over time (kind of oscilloscope chart). From the section Trend of the Options page, you can customize the report section such as: the chart size to generate, and markers to add on the chart (test name, mean, limits, 2 sigma space, 3 sigma space or 6 sigma space).



- This report section creates a scatter chart. It is the typical report you would use when analyzing the correlation between two parameters. From the regular, or indeed,  noisy shape of the chart, you can quickly diagnostic a relation between two parameters. From the section Correlation of the Options page, you can customize the report section such as: the chart size to generate, and markers to add on the chart (test name, mean, limits, 2 sigma space, 3 sigma space or 6 sigma space).



- This report section creates a Gage R&R (Box plots) chart. Very useful to visually analyze tests that are in the process or out of control, and to analyze repeatability and reproducibility . From the section Gage R&R-BoxPlot of the Options page, you can customize the chart size and sorting mode.



- This report section lists test results (Datalog) for numerical analysis. From the section Datalog of the Options page, you can customize which datalog information to show or hide (e.g: test limits, test name, die location, etc).

 

- This selection lets you interact with your data and play on the test limits until you have the Cp, Cpk you want. Once you're done with your trials, you simply click the snapshot icon so to save your table into the HTML snapshot gallery. This drilling function is quite convenient when working on guard banding and seeing the impact of test limits on the yield and Cp,Cpk statistics. This selection is maintained in the GEX-Instant Report module for historical reason, but it is the only selection that allows data interaction. For real Data Mining, checker the GEX-Data Mining, Yield Analysis module


- This report section creates a list of tests pairs that highly correlate (correlation between test computed using the Pearson's correlation formula). One typical usage of this report is to spot tests that strongly correlate (correlation score close to 1.0), and consider dropping one of the test so to reduce testing time. This report also displays the scatter plot of the test pairs so to easily confirm the correlation level and its validity.
