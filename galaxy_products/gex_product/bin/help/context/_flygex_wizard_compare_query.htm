Advanced help
More details:

: Create the queries to compare. Enter as many queries as datasets to compare [more]

: Edit a query, lets you change its definition and filters.

: Remove the selected query from the list.

Clear all: Empty the list of queries.

: Reorder the queries as the first one in the list will be considered the reference, and all drifts are computed from its dataset.

Next: Same as clicking the Settings tab. It brings you to the next assistant page to select the report sections to create and/or the interactive drill to perform.
