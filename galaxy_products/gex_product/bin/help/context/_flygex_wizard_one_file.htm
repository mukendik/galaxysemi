To analyze your file:

  1. Clickto browse your disk and select the file to process (or use Drag & Drop)

  2. Click to set the parts/binning and sites filters
    e.g.: Parts 1-10, 11, 15

  3. Click Next button

Advanced help
More details:

: Browse you disk or network and select the file to process. GEX supports many data files format such as STDF (V3 and V4), ATDF, WAT, etc. Examinator supports Drag & Drop.

: Lets you define a filter for the data extracted from your data file. You can specify a list of part numbers (e.g.: 1 to 12, 19, 25), or binning to process or ignore. You can also activate a filter on the tester site so as to only process data tested on a specific site (e.g.: site 0). From the Properties window, you also see the file header information ; it includes: file creation date/end of test date, operator name, tester name, OS name, yield results, etc.

Next: Same as clicking the Settings tab. It brings you to the next assistant page to select the report sections to create and/or the interactive drill to perform.
