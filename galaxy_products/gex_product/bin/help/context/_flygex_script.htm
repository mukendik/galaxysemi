Advanced help

Favorite scripts toolbar:

: Add a script file to your list of favorites

 : Edit the title (script description string) of the script file currently highlighted

: Edit the source code of the script file currently highlighted

: Run script file currently highlighted

: Remove selected script file from the list of favorites

 

Screen window toolbar:

: Create script from scratch: launches your text editor. When you save your edits, make sure the file format is simple ASCII, and not a proprietary wordprocessor format!

: Browse your disk/network to select a script file then run it.

: Edit the script code of the last executed script.

: Run again the current script
