Filter specific Bin #

To have Examinator ignore or process specific Bins, do the following:

  • Click the Data tab to see your list of file(s) to process

  • For each file listed, edit its properties  then in the Data Filter group, set the Process field to what you want. See table below:

Set 'Process' field to In order to...
All parts (any Bin) Process ALL parts found in your data file, regardless of the binning result.
Good parts only Process Bin1 parts only.
Failing parts Process ALL Binning except Bin1 parts.
List of Bin... Process a list of given bins.
E.g.: 2,3 would make Examinator only import data from parts with a binning of 2 or 3.
All bins except... Ignore a list of given bins.
E.g.: 5 to 9   would make Examinator import all data except parts with a binning between 5 and 9.
