Correlation between testers

Prerequisites: To compare testers, one typical process consists in testing a golden device in loop at each tester so to create a repeatability data file. For a statistically valid analysis, each golden device should be tested at least 30 times; many prefer to loop 100 times so as to ensure enough samples, and increase the statistical results accuracy.

To perform a correlation analysis between testers, do the following:

  • Click the Compare files from the home page

  • Select all the repeatability files you want to involve in your correlation analysis (usually is one file per tester).

  • Click the Next button to display the report settings page and select all the report sections you want

Note: When comparing files, the histograms report may take a while to build all the HTML pages as each test histogram has to draw and overlay data from each file. This process can take a minute or so if many tests are involved. Therefore, you may sometimes want to disable the histogram generation when focusing on other reports and looking for fastest data processing speed!
