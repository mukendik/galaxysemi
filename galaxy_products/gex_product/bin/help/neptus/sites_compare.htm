Compare data from different sites

To compare a site with other, different sites, do the following:

  • Click the Compare files from the home page

  • If all data are in the same data file, select the same file as many times as sites to compare. If you have one file per site, then simply select all the files to compare

  • For each file listed, edit its properties  then in the Data Filter group, set the Valid sites field to only include your specific site. E.g.: 3 for only importing data from test site#3.
