Import files to a Database

Quantix Examinator Pro supports many different native test data file formats and lets you import them easily into your databases:

  • ATDF [more]

  • CSV / Excel [more]

  • GDF from Agilent [more]

  • PCM from UMC foundry [more]

  • PCM from HYNIX foundry [more]

  • STDF [more]

  • WAT from TSMC foundry [more]

  • etc.

Because Examinator Pro supports the above native data formats, you do not need any parser or third-party software to process it! From the Database Admin page, simply select the files you want to import, it is that simple!
