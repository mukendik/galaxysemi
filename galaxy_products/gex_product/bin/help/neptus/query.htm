Analyze all data between two dates

Selecting files individually is fine as long as you perform engineering data analysis...but when its about production data, all you want is to manipulate data tested during a given time period, and matching  certain criteria for the tester name, lot name, program name and device type. 

Quantix Examinator includes such a feature, and all it takes is just one mouse click to launch just such an assistant!

 

Case study : Merge all data between two dates

Let's say you want all data tested over a given time period. Also, let's say you only want data tested for a given lot and for a given tester. You would then do the following:

  • Run the 'Merge files' assistant

  • Click the Smart Import ( icon)...you'll then have a dialog box that lets you define the two dates limits, the directory where STDF files are located, and filtering option such as: lot name, tester name, program name strings expected. For full details, see the Smart Import page.

  • Enter all the settings to be used by the Smart Import assistant then click ok.

  • The Smart Import assistant will scan your given directory (and sub-directories if selected) and look for all files that include data tested between your two dates, then check the filter strings. All files matching passing the Smart Import conditions will be automatically added to your list of files!

  • Click the Next button to review you report charts and tables to create, then click Build Report!

 

Case study : Compare data sets from two time periods

Let's say you want to compare data from two time periods. This may consist of tens or hundreds of files...then how can you easily define these two sets of data and locate their files? Here too, thanks to the Smart Import assistant, it only requires few mouse clicks.

You would then do the following:

  • Run the 'Compare groups of  files' assistant

  • Create your two groups that will include your two clusters of data 

  • For each group, do the following: Click the group name (so to have it selected and ensure files imported will go into that group). Click the Smart Import ( icon)...and follow the same steps as for the case study above: enter all your import settings and click ok.

  • Click the Next button to review you report charts and tables to create, then click Build Report!
