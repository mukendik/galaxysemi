Processing HYNIX PCM Parameters files

The Examinator product line supports native PCM (Process Control Monitor) files from Hynix in ROW data format and lets you manipulate any of its parameters. Examinator supports many other test data formats so you are never let down!

Also, if Examinator doesn't support your HYNIX PCM files as expected, simply forward you file to the Quantix support for further help.
