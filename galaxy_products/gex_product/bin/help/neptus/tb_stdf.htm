Convert Data to STDF format!

The Examinator ToolBox lets you convert any semiconductor data file into STDF V4 file format. As this format  is widely supported by all professional data analysis tools, you'll now be able to read and process any of your data file! Also, this STDF converter allows you to workaround third-party parsers that do not work fine or are no longer supported.

Examinator ToolBox lets you convert many data files into STDF, including:

       ATDF
       CSV / Excel
       GDF from Agilent
       PCM - UMC Foundry
       WAT - TSMC Foundry

Note: converted files have the extension .gextb.std
E.g.:
my_file.csv becomes my_file.csv.gextb.std
