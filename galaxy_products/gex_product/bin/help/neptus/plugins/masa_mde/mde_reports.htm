MASA MDE Models: MDE Reports

 

The MDE report page

From the main report page, the user can access either the standard Examinator report page or the MASA MDE report page. However, some links allow to navigate between  the mutivariate analysis results provided in the MASA MDE report page and the univariate analysis results presented in the standard Examinator report page.

The MASA MDE Report page has links to two types of reports:

  • abnormality score reports
  • wafer result reports

 

Abnormality score reports

Abnormality score histogram report

This report displays an histogram of the MDE abnormality score distribution over the analysed data set. Normal data have low scores while abnormal data have high scores. The histogram bins that contain abnormal data are shown in red. If the analyzed dataset is statsitically equivalent to the reference dataset used initially to train the MDE model, this histogram will be flat. 

A link to an interactive view of the same data is provided, which allows for a closer examination of the distribution. Another way to look at the same data is provided by the abnormality score trend report.

Abnormality score trend report

This report displays a trend chart of the MDE abnormality scores over the analysed data set. Normal data have low scores while abnormal data have high scores. Each x-coordinate corresponds to a wafer. Abnormal wafers have scores above the red line. 

A link to an interactive view of the same data is provided, which allows for a closer examination of the distribution. Another way to look at the same data is provided by the abnormality score histogram report.

Use the wafer result report and wafer summary reports to individually examine the abnormal (and normal) wafers.

 

Wafer result reports

Wafer score report

The wafer score report displays the list of wafers that have been analyzed together with their MDE abnormality scores. Abnormal scores appear on a red background in the report. 

This report provides links to the wafer summary reports of all tested wafers.

Wafer summary reports

A wafer summary report is generated for each tested wafer. The page header displays the lot ID, wafer ID and MDE abnormality score of the wafer.

The remaining of the page presents the different tests by descending order of deviation to reference (called a Z-Score, a measure of univariate abnormality). Thus, the most abnormal tests are presented first. For each test, the following information is displayed:

  • Test name: current test name
  • Min measurement: minimum measurement for the current test (recall that a test can be applied at several locations on the same wafer).
  • Max measurement: maximum measurement for the current test (recall that a test can be applied at several locations on the same wafer).
  • Z-Score: deviation between the current measurement and the distribution of values for the same test over the reference dataset used initially to train the MDE model. If the test is applied at several locations, then the location with the worst Z-Score is considered.
  • Reference mean: mean of the distribution of values for the current test over the reference dataset used initially to train the MDE model. If the test is applied at several locations, then the location with the worst Z-Score is considered.
  • Reference standard deviation: standard deviation of the distribution of values for the current test over the reference dataset used initially to train the MDE model. If the test is applied at several locations, then the location with the worst Z-Score is considered.

This report also provides links to the ''histograms of tests'' report and from there to the associated univariate analysis results.
