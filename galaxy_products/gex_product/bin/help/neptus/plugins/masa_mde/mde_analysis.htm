MASA MDE Models: MDE analysis

Activating the ''Apply MDE analysis to a Dataset'' link on the Examinator's home page allows the user to select a dataset and open the MDE model application dialog.

To apply an already generated model, choose an MDE model by clicking on the corresponding model name. By default, the last model generated
is proposed. Only the models that are in the current model library are available for application (see the MDE administration dialog).

Once an MDE analysis has been performed, the standard Examinator analyses are also applied and the user is directed to the main report page.
