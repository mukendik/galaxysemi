MASA MDE Models: About MASA

Based on leading-edge research in adaptive computing, MASA has developed core technologies that can be applied across a broad range of market sectors to produce enhanced products and deliver powerful business solutions.

MASA provides software components and software applications in three domains:

  • Behavior simulation: MASA provides software components that help model and simulate how human beings select actions and make decisions. These components are based on an internally developed kernel called DirectIA®. They are used in particular in military simulation systems, for the French army and in collaboration with Thalès.

  • Dynamic optimization: MASA provides decision support systems that help optimize complex industrial processes, using an internally developed optimization engine. The applications are bespoke and customized for each problem, so that they are close to the operational users’ needs.

  • Statistical learning: MASA provides software components that help better control industrial processes and product quality. The components are usually plugged into existing IT systems (MES) and make, for instance, automatic fault detection and classification easier.

MASA has been working with various industrial partners and clients. Among them: Thalès, Renault, DGA, COGEMA, Royal Mail, Matra, SITA, CGEA-Onyx, Ubisoft, SERNAM, EADS Telecom, La Poste, CEGOS, UPS Logistics, ST-Microelectronics, etc. More info can be found at: www.masagroup.net

Today, the dynamic optimization engine and statistical learning components constitute the kernel of the BlueKaizenTM Software Suit.
