MASA MDE Models: Training an MDE Model

Activating the ''Generate an MDE Model'' link on the Examinator's home page allows the user to select a reference dataset and open the MDE training dialog.

To train an MDE model just enter a new name for the model in the ''Model name'' field. All model names must be unique within the same model library, so an error message will be displayed if the model name chosen has already been given to an existing model.

You may also choose to supply a rejection rate in the ''Rejection rate'' field different from the default value (2.00%). The rejection rate must be a real quantity in the range [0.00%-100.00%].

The training procedure lasts a few seconds. After training, the resulting MDE model is stored in the MDE model library so that it can later be used to perform an MDE analysis.
