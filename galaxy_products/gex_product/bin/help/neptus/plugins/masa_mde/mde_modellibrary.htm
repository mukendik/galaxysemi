MASA MDE Models: The MDE Model Library

Whenever an MDE model is created using the MDE model training dialog, it is automatically stored in the MDE model library.

Activating the "Model Admin" link on the Examinator's home page opens the MDE administration dialog.

 

Looking at the list of available MDE models

Upon opening the MDE administration dialog, the list of available MDE models in the current MDE model  library is displayed, along with the following information for each model:

  • Name: name given to the MDE model by the user at training time
  • Type: type of the MDE model
  • Generated on: date of generation of the MDE model
  • Rejection rate: value of the MDE model's rejection rate parameter
  • # Samples: number of sample wafers used to train the MDE model
  • Size: size of the model image file that contains the MDE model information

 

Deleting an MDE model

To delete a model, select a model in the model list and press the Delete button. Deleting a model amounts to removing the entry of  the model in the index file and erasing the model image file.
