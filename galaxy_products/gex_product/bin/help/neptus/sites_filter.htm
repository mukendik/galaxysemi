Filter data from a specific site#

To have GEX only process data from a specific test site, do the following:

  • Click the Data tab to see your list of file(s) to process

  • For each file listed, edit its properties  then in the Data Filter group, set the Valid sites field to only include your specific site. E.g.: 3 for only importing data from test site#3.
