Editing test data

The Examinator ToolBox lets you edit any semiconductor data file. To allow easy editing, the data file is first converted into a Excel type table so you can easily edit any cell. Examinator ToolBox lets you edit many data files, including:

       ATDF
       CSV / Excel
       GDF from Agilent
       PCM - UMC Foundry
       STDF
       WAT - TSMC Foundry

 

This editor allows unlimited number of lines and columns to be visualized. Unlike Excel, Examinator ToolBox doesn't suffer of any file size or column index limitation!
