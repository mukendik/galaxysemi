Printing report pages

All you need to do to print a report page is to click the printer icon ( icon) in the toolbar,  then select the printer device where the report page currently displayed is to be sent. As charts may be quite large you should print selecting the Landscape orientation.

Hint: If you have installed the Adobe Acrobat package, you can select the PDFWriter or Acrobat Distiller printer device to create a PDF file!

Page breaks issues or image truncated?

When printing, images may be truncated (such as Histograms) if your printing page is too small. You can fix this setting the printer format to Landscape orientation, or also pre-process the HTML pages under Microsoft Word.
