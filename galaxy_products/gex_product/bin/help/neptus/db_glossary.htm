Glossary

Parameter Vs. Test...no difference!

A database includes records of parameters. Queries are executed on the database to retrieve values for selected parameters. Under Examinator Pro, a parameter is called a test. So Parameters and Tests have exactly the same meaning. "Parameter" is a more generic terminology, while "test" is more specific  to the semiconductor industry...but then, this is Quantix's focus!

Most of the parameters you'll manipulate using Examinator Pro will be real tests (from metrology instruments, inspection instruments, Electrical test, Wafer sort, final test to assembly and more)...so the terminology "test" should not shock anyone! It should,  rather,  make the queries more familiar

Of course, this is only a terminology issue, and this doesn't prevent you manipulating any type of data you want. Examinator Pro lets you import data files where parameters can range from test values, to temperature, to price or any other numerical information. Examinator Pro gives you full flexibility to import any numerical values in its database.
