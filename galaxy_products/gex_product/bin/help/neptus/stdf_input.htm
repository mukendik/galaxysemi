Processing STDF test data

Quantix Examinator supports test data files in native STDF V3 and STDF V4 formats. Most of the ATE testers have the capability to output test results under this format. Examinator supports many other test data formats so you are never let down!

Because Examinator supports the native STDF data formats, you do not need any parser or third-party software to process it! Simply select the files you want to process, it is that simple!
