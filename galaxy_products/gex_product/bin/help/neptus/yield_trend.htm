Plot Yield/Binning trend over lots

To see your production yield level, all you need to do is select all the data files you want, then trend the relevant binning (typically Bin1). Below is an example of trend chart with details and Yield information in a table. As multiple lots are merged, each lot starts with a vertical marker showing the lot name:

 

You can do the same and even more! Simply do the following:

  • Select the Merge files from the Examinator home page  

  • Pick all the production files you want to merge into the Trend report

  • click the button Next to see the "Settings" page. Go to the "More Reports" field and select Trend control chart

  • Select the trend chart to be over Software binning (or hardware binning, depends what you want)

  • In the edit field, enter the Binning# to trend, or a list of binning if you want to combine multiple binning in the trend chart (e.g.: 1 to 10)

  • Click the button Build report and see your trend chart created in seconds!

Master all the power of the trend chart:

Tip: If you want to compare groups of files (e.g.: comparing two lots), and do not care about other reports, then disable the Test Statistics and Histograms reports generation: this will dramatically boost the report generation speed!
