Report in WORD format

Examinator (Pro Edition, PC version only) allows to create your report as a MS-Word document. It automatically creates all reports section into a fine word document with page breaks and table of indexes! Paper size (US Letter or European A4) can be configured from the 'Options' tab, section Report output / paper size.

Hint: You can customize the charting style and markers from the 'Options' tab for any of the charts, including: histograms, trends, wafer maps, etc. Only image size is not configurable as Examinator automatically adjusts image to best fit your page size with appropriate page breaks.

To select the report output format to WORD, either select the report format from the 'Settings' page just before building the report, or click the Options tab from the application, then go into the Report Output / Output format section
