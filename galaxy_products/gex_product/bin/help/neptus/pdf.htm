Creating .PDF files

If you are using Examinator Pro, you can directly create reports in PDF format:

From the 'Options' tab, select Report output/ Output format = PDF , or select the PDF output format from the 'Settings' page, just before building the report. Under this mode, Examinator automatically adapts report images and tables to exactly fit into page for perfect printing and page breaks!

Alternatively (and if you don't have Examinator Pro), to create a PDF file from a report page displayed, you need to have the Adobe Acrobat package installed. Then, proceed as if you would like to print the page (click the printer icon in the Examinator toolbar). Last, select the printer name: PDFWriter or Acrobat Distiller so the report is printed into the Adobe PDF file generator. It's that simple!
