Report Generation

This task lets you execute a query file (the script file as created by the Examinator application when you click the Build Report). The HTML report created can be sent by email or posted to the intranet. Multiple emails can be entered if separated with the semicolon character.
 

Where is the report created?

Yield-Man always creates such reports into its database path (as defined under the 'Options' tab) into the following folder name:

<database>/report.mo/<time.stamp>/
 

Report Format

If the report is to be sent by email, do not select HTML as the output format as only the report home page would be sent! Instead, favor the other formats that make the full report available under one single file (e.g.: PDF format).
For reports to be sent by email ,favor the Excel/CSV or PDF format that are the most compact compared to Word and PowerPoint

 

Unix platform

If you are running Yield-Man under Unix, then Word and PowerPoint formats are not supported and if your script is requesting such formats, the report will be created in PDF instead.
 

How to create a script?

  • Manually create one (see on-line help Scripting section)
    or

  • See scripts templates available on Quantix's web site support section or on-line help.
    or

  • Use Examinator-Pro to run a query ; then click the save script icon from the report home page. You can also click the script icon in the toolbar then click the "Edit last script executed" iconin the script window console (right pane) to see the script executed.

Mailing list

The email report/notification is sent to the email addresses or mailing list file you define. If you specify email addresses, use the semi-colon separator (e.g: user1@galaxysemi.com ; user2@galaxysemi.com)

If you click the Mailing list icon to specify a mailing list file, make sure the file selected complies with the following format:

  • File is in ASCII format

  • Each line includes one email address in format name@domain

  • If multiple addresses appear on one line, use the semi-colon separator ;

Example of mailing list ASCII file:

user1@galaxysemi.com
user2@galaxysemi.com
user3@galaxysemi.com ; user4@galaxysemi.com
user5@galaxysemi.com
 

Note: Have your system administrator check that the Monitoring email service is correctly setup otherwise emails notification won't go!
