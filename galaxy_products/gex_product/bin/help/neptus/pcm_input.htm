Processing PCM Parameters files

The Examinator product line supports native PCM (Process Control Monitor) files from UMC in ROW data format and lets you manipulate any of its parameters. Examinator supports many other test data formats so you are never let down!

About the ROW data format: All results for a given parameter are listed in the same row. Last lines of the file include the statistical information (max, min, average...) and specification limits. As UMC can provide you with ROW or COLUMN PCM files, make sure to request files in ROW format.

Because Examinator supports native the PCM data format, you do not need any parser or third-party software to process it! Simply select the files you want to process, it is that simple!

To comply with the Examinator wafermap coordinates system, each PCM site is assigned a wafer map die area using X,Y coordinates as shown below:

PCM site name PCM site # Examinator X,Y die location assigned
Center 1  X= 1, Y= 1
Bottom 2  X= 1, Y= 2
Left 3  X= 0, Y= 1
Top 4  X= 1, Y= 0
Right 5  X= 2, Y= 1

For site# higher than 5, Examinator forces X,Y die location to 0.
