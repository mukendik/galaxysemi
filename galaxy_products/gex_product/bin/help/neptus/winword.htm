Report in Word format

HTML is a nice and easy to share format. However creating presentations and adding your comments and other information is much easier with a text editor such as Microsoft Word.

Import HTML pages

To import any HTML page, simply follow this sequence:

  • Launch Microsoft Word and create your document template

  • From the menu bar, select the menu item Insert then the submenu File

  • From the browsing dialog box, select File of type: All word documents *.htm *.doc,...

  • Browse your disk to select the HTML page to import

  • Microsoft Word will then prompt you to select the correct file converter filter, 
    select: 

    HTML document.

 

Import Screenshots

Using the Examinator solution toolbar Snapshot / Copy icon , you can copy the Examinator window's content into the clipboard. This copy is an image you can then paste into any other applications (MS-Word, PowerPoint, Outlook, etc.). To capture the Examinator window's content and paste it into another application, simply follow this sequence:

  • Launch Examinator, and build your report

  • Browse and scroll into the report pages until you have the Examinator window's content include the exact image you want to capture (resize Examinator if you want).

  • From the Examinator toolbar, click the Snapshot / Copy icon

  • From any other application (MS-Word or Outlook or Powerpoint, etc.), select the Paste option (usually the Edit / Paste menu selection)


Tips using Word

  • Set the Page orientation to Landscape: this makes all images such as histograms to fit well on screen.

  • Insert page breaks after each report section, and after each chart (use the Word Replace command to replace at one go a given header string with the same string preceded by a form feed!).

  • Note: MS-Word does not import the images from your HTML files but only the links to them. So if you email the resulting Word file, all images will be missing, and only text and tables will remain!...for email exchange, best is to compress your HTML folder with all your reports, or create a PDF file if no edits are needed, or cut/paste screenshots using the Snapshot / Copy icon in the Examinator toolbar. Word format is convenient if you want to create a report and or print it.
