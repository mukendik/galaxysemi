Connecting to Corporate Databases (Oracle, mySQL, ...)

The Examinator Pro solution lets you deal with data collections (folders) local to your machine, or on a server. This gives you full flexibility to create and maintain private data collections for your own use (local databases) and also share more generic data collections that are stored on a server (server databases)

Local Databases

The local databases are local folders you can administrate from the Examinator Pro (Create, Delete, Import files). These folders are created in the Examinator Pro databases directory (unless changed from the Options tab).

Under Unix, each local database you create is a folder you'll find under the following path:

$HOME/galaxy_examinator/databases

Under Windows, the same local database will be seen under the following path:

<Examinator Install Folder>/databases

Example: See how the two local databases simulation-1 and Bench3 data would be saved on your system.

<Examinator path>/databases/simulation-1
<Examinator path>/databases/Bench3_data

 

Server Databases

The server databases work exactly the same way as for local ones. You may Create/Delete server databases and import files to them if you have write permissions to the relevant server folders (this is you and your system administrator to decide about this!). Names and path conventions are the same as for local databases as well.
