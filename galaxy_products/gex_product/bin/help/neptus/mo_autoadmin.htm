Auto Admin & Cleanup

This task lets you defined few auto admin settings so Yield-Man performs all the housekeeping for you! Settings include:

  • Auto admin starting time: this lets you define at what time the daily housekeeping must be triggered (default is 11PM).

  • Erase reports on server: Lets you define how long to hold reports posted to the server (these are the reports created on yield exceptions or auto reporting). Default is 1 week.

  • Sys. admin email: lets you define the administrator email address where low level error messages are to be sent. This field is optional.

Note: Only one Auto admin task allowed in the Yield-Man list of tasks.
