Filter specific Parts #

To have Examinator ignore or process specific parts, do the following:

  • Click the Data tab to see your list of file(s) to process

  • For each file listed, edit its properties  then in the Data Filter group, set the Process field to what you want. See table below:

Set 'Process' field to In order to...
All parts Process ALL parts found in your data file
All parts except... Ignore a list of given parts.
E.g.: 5 to 9   would make Examinator import all parts except parts 5 to 9.
List of parts Process a list of given parts.
E.g.: 1 to 10   would make Examinator only import data from parts 1 to 10.
