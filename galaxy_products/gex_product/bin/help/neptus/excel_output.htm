Report in Excel / CSV format

The Excel / CSV output format is convenient if you want raw information rather than Examinator charts. The CSV format is an ASCII file format with ',' (comma) characters used as delimiters.

To select the report output format to EXCEL / CSV, simply click the Options tab from the application, then go into the Report Output / Output format section

Note: Advanced reports (except Datalog) are not supported in CSV format.
