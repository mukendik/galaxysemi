Create PowerPoint(tm) presentations!

Microsoft PowerPoint is one of the best tools around when it comes to designing presentation slides with Examinator screenshots. For long reports that include many Examinator report pages, then instead use Microsoft Word as it is a better tool for importing a whole HTML page (see details).

Import Screenshots

Using the Examinator solution toolbar Snapshot / Copy icon , you can copy the Examinator window's content into the clipboard. This copy is an image you can then paste it into any other applications: PowerPoint, or any other application that supports the cut/paste function (e.g: MS-Word, Outlook, Paint, Photoshop, etc.). To capture the Examinator window's content and paste it into PowerPoint, simply follow the steps:

  • Launch Examinator, and build your report

  • Browse and scroll into the report pages until you have the Examinator window's content include the exact image you want to capture (resize Examinator if you want).

  • From the Examinator toolbar, click the Snapshot / copy icon

  • From Powerpoint, select the Edit / Paste menu selection.

  • Manipulate the image the way you want: resize, add border, set animation, etc.


Tips

  • From the File/Page setup menu item, set the Page orientation to Landscape: this makes all images such as histograms to fit well on screen.

  • You can copy any page visited under the Examinator environment: this includes the report pages, but also all other pages such as the online help, the assistant pages, etc.
