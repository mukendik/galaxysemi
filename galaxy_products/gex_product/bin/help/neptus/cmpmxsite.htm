Compare multi-sites data

Regardless of whether the test results for all sites are in a single file, or in separate files...the process remains the same!

 

Case study : One file to process

To have Examinator compare tests of each site you want, you will do the following:

  • Run the 'Compare files' assistant

  • Select the list of files to analyze ( icon)...you can select the same file multiple times, once for each site to compare!

  • Double click each file name to display its properties and parts/site filter: select the site filter

  • Click the Next button to review you report charts and tables to create, then click Build Report!
