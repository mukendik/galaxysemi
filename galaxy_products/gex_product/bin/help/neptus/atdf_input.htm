Processing ATDF test data

The Examinator product line supports test data files in native ATDF format. This is an ASCII format usually supported by testers that do not have the capability to create files in STDF format. Examinator supports many other test data formats so you are never let down!

Because Examinator supports the native ATDF data format, you do not need any parser or third-party software to process it! Simply select the files you want to process - it's that simple!
