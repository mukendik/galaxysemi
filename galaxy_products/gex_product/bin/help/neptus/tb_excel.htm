Convert Data to Excel/CSV!

The Examinator ToolBox lets you convert any semiconductor data file into a ASCII Excel/CSV table with tests/parameters listed in columns. With STDF, this format  is one of the most common standard format supported by all professional data analysis tools. You'll now be able to read and process any of your data file! Also, this converter allows you to workaround third-party parsers that do not work fine or are no longer supported.
 

Examinator ToolBox lets you convert many data files into Excel/CSV, including:

       ATDF
       GDF from Agilent
       PCM - UMC Foundry
       STDF
       WAT - TSMC Foundry


As the Excel/CSV file generated may include more columns that Excel support, we suggest that you use the Examinator ToolBox internal Editor to Edit the Excel/CSV file, Examinator doesn't suffer of the Excel limitations!

Note: converted files have the extension .gextb.csv
E.g.:
my_file.std becomes my_file.std.gextb.csv
